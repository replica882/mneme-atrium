导入你和 AI 的对话历史（ChatGPT / Claude），在花园里继续聊天。AI 会记住你说过的话，你决定它怎么说话。

你的数据只在你的设备上。没有我们的服务器，没有第三方中转。

## 核心能力

| 能力 | 描述 |
|------|------|
| **对话浏览** | 树状分支地图、全文搜索、日历时间线、收藏与标签 |
| **实时聊天** | 15 家内置 API 提供商 + 自定义，流式输出，附件 / 图片 |
| **AI 记忆** | 自动从对话提取事实，按冷热衰减，按预算注入下次对话 — AI 记得你 |
| **人格系统** | 酒馆式 Prompt 插槽 + 采样参数 + 角色卡（V2/V3）+ 世界书 — 你决定 AI 怎么说话 |
| **联网搜索** | 13 家搜索后端任选，AI 可查网、读网页 |
| **文件库** | 内置 Markdown 编辑器（实时预览）、HTML 预览、迷你浏览器 |
| **贴纸** | 图片贴纸 + 便签 + 画板，给对话留下你的标记 |
| **有声书** | 导入音频，自动续播，书签笔记，后台播放 |
| **归档** | Markdown 导出、时间机器式自动备份 |

## 系统要求

- macOS 15 (Sequoia) 或更高
- iOS 26 或更高（未来会兼容 18）
- 需要自备 API Key（OpenAI / Anthropic / DeepSeek / OpenRouter 等）

## 隐私

- 所有数据存储在本地设备
- API Key 存储在系统钥匙串
- 不采集任何用户数据
- 不经过任何我们运营的服务器
- 可选 iCloud 同步走 Apple 系统级通道（你自己的账号）
