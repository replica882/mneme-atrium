# acquaint

---

## 1. 一句话本质

acquaint 是派对上那个把你拉过去说"来，我给你们介绍一下"的人——它只管让你和某样东西**照个面**，至于你们后来成不成深交，它不管。

---

## 2. 词源考古

### 骨骼拆解

```
acquaint
  ← ac- (= ad-, 朝向、去往)
  ← quaint? ——不。这里的词根不是 quaint（古雅的）。
  ← 来自 Old French: acointier（使认识、使熟悉）
  ← Medieval Latin: accognitāre（使知道）
  ← Latin: ac- (ad-) + cognitus（已知的），cognoscere 的过去分词
  ← cognoscere = co-（一起）+ gnōscere（知道）
  ← PIE 原始印欧语根: *ǵneh₃-（知道）
```

这个 **\*ǵneh₃-** 是英语里"知道"家族的总族长。它的后代遍地开花：

| 后代 | 怎么来的 |
|------|---------|
| **know** | 日耳曼一脉直传 |
| **cognition** | 拉丁 cognoscere → 认知 |
| **recognize** | re- + cognoscere → 再次知道 → 认出 |
| **connoisseur** | 法语继承了 cognoscere → 鉴赏家（"真正懂行的人"） |
| **incognito** | in- + cognitus → 不被知道 → 隐姓埋名 |
| **reconnaissance** | 再次去认知 → 侦察 |
| **diagnose** | dia-（穿透）+ gnōsis → 透过表象去知道 → 诊断 |

所以 acquaint 的骨子里就是 **"让你朝着'知道'走一步"**。

### 历史流变

**13世纪**：从古法语 *acointier* 进入中古英语，写作 *aqueinten*。在古法语里这个词的意思很宽——从"介绍认识"到"与……亲密"（甚至有性暗示的用法）都有。当时法语里 *acointier* 的意思更接近"和某人交好"。

**14-15世纪**：英语里的含义开始**收窄**。那个"亲密""深交"的义项逐渐退场，固定在"初步认识、使了解"这个浅层含义上。这很有意思——一个从法语借来时暗含温度的词，在英语里被冷却了，变得更客气、更保持距离。

**16世纪后**：进一步正式化。日常口语越来越倾向于用 *get to know* 替代它，acquaint 退守到书面语和正式场合。

**一个有趣的副产品**：*quaint*（古雅的、别致的）和 acquaint 在现代英语里看着像是亲戚，但它们的命运完全不同。quaint 也来自古法语 cointe（来自 cognitus），但它走了另一条路——从"有学问的"→"巧妙的"→"奇特精致的"→"古色古香的"。同一个拉丁祖先，两个完全不同的人生。

## 3. 故事 / 画面

一栋大楼的门厅。

有个穿制服的人站在旋转门内侧，每次有新客人进来，他就上前一步：

"这是前台，那边是电梯，三楼有会议室，地下一层是停车场。"

三十秒。说完了。他退回原位，等下一个人。

他不会带你去三楼坐下来开完整场会议。他不会陪你在地下停车场找到车位。他甚至不会记住你的名字——因为他的工作就是那三十秒的引路。他让你**不再是个对这栋楼一无所知的人**，但你离"熟悉这栋楼"还差得远。

这就是 acquaint 的全部工作范围：**从"完全不知道"到"大概知道"的那段路程，而且仅限于那段路程。**

## 4. 多面体：核心义项展开

### 义项 A：使（某人）了解、使熟悉

最常见、最核心的用法。几乎总是 **acquaint sb with sth** 的结构。

> *Let me acquaint you with the procedures before your first day.*
> 在你入职第一天之前，让我给你介绍一下流程。

和"一句话本质"的关系：门厅引路员在介绍楼层分布——让你对一个陌生事物"照个面"。

### 义项 B：be/become acquainted with — 与……相识；对……有所了解

被动态用得比主动态还多。**be acquainted with** 描述的是一种状态：你认识了，但别指望有多深。

> *I'm acquainted with his work, but I wouldn't call myself an expert.*
> 我对他的研究有所了解，但谈不上专家。

和"一句话本质"的关系：你已经被领进了门厅，你见过楼层平面图，但你没在里面住过。

### 义项 C：acquaintance — 认识但不深交的人

名词形态。这是 acquaint 最有文化穿透力的衍生。

> *He's not really a friend — more of an acquaintance from college.*
> 他算不上朋友，只是大学时认识的人。

和"一句话本质"的关系：门厅引路员和你打过招呼，你知道他的脸——但你连他叫什么都不确定。acquaintance 就是停留在门厅层面的人际关系。

### 义项之间的统一性

三个义项指向同一件事：**知道，但只是知道了第一层。** acquaint 永远在"完全陌生"和"真正熟悉"之间的那条浅水区里工作。它不渡你到深水区。

## 5. 裂缝标注

### 常见误用场景

**❌ 用 acquaint 表示"深入了解"**

> ~~I've been acquainting myself with quantum physics for ten years.~~

学了十年量子物理，那你早就不是 acquaint 的阶段了。应该说 *I've been studying / immersed in quantum physics for ten years.* acquaint 只管开头那一小段。

**❌ 漏掉介词 with**

> ~~Please acquaint me the new policy.~~
> ✅ *Please acquaint me **with** the new policy.*

acquaint 几乎永远需要 with。这是一个不可拆卸的介词搭配。

### 易混近义词辨析

| 词 | 区别 |
|----|------|
| **familiarize** | 和 acquaint 含义最近，但 familiarize 暗示一个更主动、更系统的过程。*Familiarize yourself with the manual*（把手册读透）vs *acquaint yourself with the layout*（大概看看布局）。familiarize 期望你最终"真的熟了"，acquaint 没这个期望。 |
| **introduce** | introduce 更侧重**人与人之间**的首次认识（*Let me introduce you to my colleague*），或者引入一个新概念。acquaint 侧重"使了解情况"。introduce 是那个拍肩膀的动作，acquaint 是拍完肩膀之后的三分钟速览。 |
| **inform** | inform 是传递**特定信息**（*I need to inform you that the meeting is cancelled*）。acquaint 是传递**整体概况**。inform 像发一条短信，acquaint 像带你走一圈。 |

### 中文母语者陷阱

**"认识"的宽度陷阱**：中文里"我认识他"可以指点头之交，也可以指老友。但英文里如果你说 *I'm acquainted with him*，听者会明确理解为"不太熟"。如果你想表达"我和他很熟"，用 acquainted 反而会让对方觉得你在故意拉距离。

**"了解"的深度陷阱**：中文"我了解这件事"可以指深度理解。但 *I'm acquainted with the matter* 只是"我知道有这么回事"——远没有"了解"那么深。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境 |
|------|------|
| **acquaint oneself with** | 最常见的主动用法。"自己去熟悉一下。" *Take some time to acquaint yourself with the system.* |
| **be/become acquainted with** | 状态描述。*We became acquainted through mutual friends.* |
| **get acquainted** | 稍微不那么正式的变体。*Let's get acquainted before we start working together.* |
| **well-acquainted** | 比 acquainted 更深一层，但仍然不到"亲密"。*She's well-acquainted with the challenges of the industry.* |
| **acquaint sb with the facts/details/situation** | 公文/正式场合的常用句式。"把情况跟他说明一下。" |
| **passing acquaintance** | 非常浅的认识。*I have a passing acquaintance with French literature.*（略知一二） |
| **casual/nodding acquaintance** | 见面点头的关系。连名字可能都叫不全。 |

### 语域标注

**偏正式 / 书面。** 在日常口语中，acquaint 已经显得有些端着了——你不太会在朋友聊天时说 *Let me acquaint you with my cat*（这听起来像你在给猫写正式介绍信）。口语中的替代品是 *get to know*、*fill you in*、*give you a rundown*。

但在**商务邮件、法律文书、学术写作**中，acquaint 活得很好。它自带一种克制、礼貌的距离感，正适合这些场合。

### 活用示范

> *"I see that you are well acquainted with my library and its contents."*
> —— 柯南·道尔笔下的福尔摩斯

福尔摩斯式的 acquainted——不是随便翻过，是认真研究过但保持着一种冷静的措辞。用 *familiar* 就少了那份矜持。

> *It was during the war that I first became acquainted with fear — not the fear you read about, but the kind that sits in your stomach and stays.*

这里 *became acquainted with fear* 有一种黑色幽默：把"恐惧"当作一个你被正式介绍认识的人，好像有人在战场上拍你肩膀说"来，认识一下恐惧先生"。这种冷冰冰的礼貌和恐惧本身的剧烈形成对比，是 acquaint 被"活用"的典型效果。

## 7. 跨语言映射

**中文"相识"**最接近 acquaint 的语义窗口——"相识"本身就暗含"认识了但不深"的意味。"我们是老相识"已经需要加"老"字来加深程度，说明"相识"的默认值就是浅层的。

**法语 connaître vs savoir**：法语把"知道"分成了两个词——*connaître*（对人/地方/事物的熟悉性认识）和 *savoir*（对事实/技能的掌握）。acquaint 对应的恰好是 *connaître* 的浅层状态：*faire connaissance*（结识）。这不奇怪——acquaint 本来就是从法语借来的。

**日语「知り合い」**（shiriai）：和 acquaintance 高度对应。日语的人际距离词汇比英语更精细——「知り合い」→「友人」→「親友」（认识的人 → 朋友 → 挚友），这条梯子上，acquaintance 就站在第一级。
