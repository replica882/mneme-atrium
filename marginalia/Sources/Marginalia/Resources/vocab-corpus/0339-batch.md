# batch

## 1. 一句话本质

batch 是面包房里一炉一炉出的面包——不是一个一个做，而是**攒够一拨，一起处理**。核心不是"很多"，是"打包成一批、统一动作"。

---

## 2. 词源考古

**词根拆解：**

batch 不可再拆。它是个光杆词根，没有前缀后缀。

**历史流变：**

- **古英语 *bæcce***（一炉面包），直接来自动词 *bacan*（to bake，烤）。最初的意思极其具体：面包师把一团团面团放进烤炉，**一炉能烤多少就是一个 batch**。
- **15 世纪**：含义开始从面包房往外扩散。任何"一次性处理的一组东西"都可以叫 batch——一批信件、一批货物。关键转折：**从"一炉"变成"一批"**，烤炉消失了，"统一处理"的抽象结构留下来了。
- **20 世纪中叶**：计算机时代给了它第二次生命。IBM 大型机时代，程序员把一堆任务攒起来，排队一次性让计算机处理——**batch processing**（批处理）成了计算机科学的基础概念。这个词从面包房跳到了机房，但底层逻辑一模一样：攒够了，一起上。

**文化地层：**

batch 的语义漂移非常干净：从具体（一炉面包）→ 日常泛化（一批任何东西）→ 技术术语（批处理）。每次扩展都没丢掉"一组东西被统一处理"这个骨架。这在英语里算难得的——很多词漂移后面目全非，batch 倒是一路忠于自己。

它和 bake 是亲兄弟，但现代英语使用者几乎感受不到这层血缘了。

---

## 3. 故事 / 画面

凌晨四点的面包房。

师傅不是做好一个面包就开一次烤炉——那太蠢了，烤炉预热要半小时。他把 30 个面团揉好，码在烤盘上，等最后一个面团就位，**啪**，一起推进炉膛。

这就是 batch 的全部哲学：**逐个准备，统一执行**。

现在把面包房换成你的邮箱。你不会收到一封邮件就回一封——你攒到下午三点，泡杯咖啡，**一批全回了**。或者洗衣机：你不会一只袜子洗一次，你等脏衣篓满了，**一批塞进去**。

batch 的对立面是 real-time（实时）。实时是消防员——来一个处理一个，等不了。batch 是邮差——攒一袋，统一投递。

## 4. 多面体：核心义项展开

### 义项一：一批/一组（最基础）

> *We received a fresh **batch** of applications this morning.*
> 今早我们收到了一批新的申请。

和"一句话本质"的关系：一组被归在一起、等待统一处理的东西。面包房的一炉面包。

### 义项二：批处理（计算机/技术）

> *The server runs a **batch** job every night to clean up expired sessions.*
> 服务器每晚运行一个批处理任务来清理过期会话。

和"一句话本质"的关系：攒一堆任务，统一执行。面包房逻辑的数字翻版。注意 batch 在技术语境里既是名词也是形容词（batch job, batch processing, batch mode）。

### 义项三：一批（带有质量/特征的）

> *That last **batch** of cookies was the best one yet.*
> 上一批饼干是到目前为止最好吃的。

和"一句话本质"的关系：同一次制作出来的一组东西，共享同一个"出身"。同一炉出来的，品质相近。这个义项在制造业里特别重要——**batch number**（批次号）追踪的就是"这组产品是同一次生产出来的"。

### 义项四：动词——分批处理

> *We **batched** the orders by region to reduce shipping costs.*
> 我们按地区把订单分批处理，降低运费。

和"一句话本质"的关系：把散的东西打成包，统一动作。动词用法是名词的自然延伸。

**底层统一性：** 四个义项共享一个结构——**一组东西因为被归在一起而获得了集体身份，等待或正在被统一处理**。不管是面包、代码任务、申请表还是饼干，batch 关心的永远是"这一拨"。

## 5. 裂缝标注

### 常见误用场景

- **把 batch 当 bunch 用**：bunch 强调的是"一堆、一串"（物理聚集感），batch 强调"同一次处理的一组"。*a bunch of grapes*（一串葡萄）不能换成 *a batch of grapes*，因为葡萄不是被"分批处理"的。
- **batch 和 lot 的混淆**：在日常英语里 *a lot of* 只是"很多"，但在制造业/质量控制领域，lot 和 batch 几乎同义（都指"一批次"）。日常语境别混。

### 易混近义词辨析

| 词 | 核心区别 |
|---|---|
| **bunch** | 物理聚集：a bunch of keys（一串钥匙）。没有"统一处理"的含义 |
| **group** | 最泛化的"一组"，没有"处理"含义，也没有"同源"含义 |
| **set** | 强调完整性和配套：a set of tools（一套工具）。batch 不关心完整不完整 |
| **lot**（技术义） | 在制造业和拍卖行里和 batch 近义，但 lot 更偏"划出来的一份" |

### 中文母语者陷阱

- 中文的"批"和 batch 高度对应（"一批货""分批处理"），这是少数中英直译非常安全的词之一。但要注意：中文说"一批人"很自然，英文里 *a batch of people* 听起来**非常奇怪**——把人叫 batch 有一种"工厂流水线处理人"的冷感。说人用 group / crowd / cohort。
- 中文"批量"对应 *in batches* 或 *batch (adj.)*，但"批评"的"批"和 batch 毫无关系——不要建立错误联想。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境 |
|---|---|
| **a batch of** + 名词 | 最基础结构：a batch of emails / cookies / data |
| **batch processing** | 计算机术语，对比 real-time processing |
| **batch job** | 定时或排队执行的自动化任务 |
| **batch number / batch code** | 制造业追溯：产品包装上常见 |
| **in batches** | "分批地"：*Process them in batches of 50.*（每 50 个一批处理） |
| **batch file** | Windows 的 .bat 文件，一组命令的脚本 |
| **the latest/first/next batch** | "最新/第一/下一批" |

### 语域标注

batch 是个**全语域词**。日常对话、技术文档、新闻报道、工业标准里都活得很好。它既不高冷也不口语化，属于中性工具词。

唯一要注意的语域差异：在技术/编程语境里 batch 几乎是术语（batch processing, batch mode），在日常语境里它更轻松（*I made a batch of brownies*）。

### 活用示范

> *Monday mornings I batch all my emails — zero inbox by lunch, zero guilt by Tuesday.*
> 周一早上我把所有邮件批量处理——午饭前清零收件箱，周二前清零内疚感。

这里 batch 被当动词用，带着一种"高效人士的时间管理哲学"的味道。现代生产力圈子里 **batching**（攒起来一起做）已经是个小热词了。

> *Each batch of recruits learned the lesson the hard way.*
> 每一批新兵都是用笨办法学会的。

---

## 7. 跨语言映射

- **中文"批"**：罕见的高精度对应。"一批货"= a batch of goods，"分批"= in batches，"批处理"= batch processing。中文母语者在这个词上有天然优势。
- **日文「ロット」(lotto)**：日本制造业直接借了英文 lot 而不是 batch，有趣的是日常日语里没有一个和 batch 完美对应的和语词——说明"分批处理"这个概念在日语里是靠外来词填补的。
- **德文 *Charge* 或 *Stapel***：德语制造业用 *Charge*（借自法语）表示"一批次"，而计算机领域用 *Stapelverarbeitung*（字面意思是"堆叠处理"）来翻译 batch processing——用"堆"的意象代替了"烤"的意象，殊途同归。
