# available

---

## 1. 一句话本质

**available** 是一扇没上锁的门——东西就在那儿，而且此刻你伸手就能够到。不是"存在"，是"存在 + 你能拿到"。

---

## 2. 词源考古

### 词根词缀拆解

```
a- (加强前缀) + vail (= val-，"有力量/有价值") + -able (能够被…的)
```

- **vail / val-**：来自拉丁语 **valēre**，意思是"to be strong, to be worth"。这个词根你已经见过很多次了——**value**（价值）、**valid**（有效的）、**prevail**（胜出）、**equivalent**（等值的）都是它的后代。
- **a-**：这里是古法语前缀，起加强/方向作用（类似 ad-），不是否定。
- **-able**：能够被……的。

所以拆到底：**"能够被有效使用的"** → **"可以拿来用的"**。

### 历史流变

| 时间 | 事件 |
|------|------|
| 拉丁语 | *valēre*（有力量、有价值） |
| 古法语 14c | *availlable*（有利的、有效的），从动词 *availler*（to avail，有用处）派生 |
| 中古英语 15c | 进入英语，最初含义是 **"有效力的、有利可图的"**（法律/商业语境） |
| 16–17c | 含义漂移：从"有效力的"→"可以被利用的"→**"可以获取的、在手边的"** |
| 现代英语 | 核心义稳定为"可获得的、有空的"，但底层的"有效力"DNA还活着 |

### 文化地层

关键漂移发生在 16 世纪：available 从描述**事物的内在属性**（"这份遗嘱是 available 的"＝有法律效力的）变成了描述**事物与人之间的关系**（"这个座位是 available 的"＝你能坐）。

这个转折很有意思——它从"东西本身有力量"变成了"东西对你有力量"。**主语从物转向了人与物的连接**。这也是为什么今天 available 既能修饰物（seats available）也能修饰人（I'm available）。

## 3. 故事 / 画面

一家深夜便利店。凌晨三点，街上什么都关了，你饿得胃疼。然后你看到那个发光的"OPEN"牌子。

门推得开。货架上有东西。收银台后面有人。

**这整个场景就是 available 的全部含义。**

不是说便利店"存在"（exist）——那条街上所有关了门的店也存在。不是说它"充足"（abundant）——货架可能半空。它的意义精确地落在这一点上：**你需要的东西，此刻，可以被你获得。** 门是开的，灯是亮的，手伸得到。

当一个人说 "I'm available"，他就是在亮那个 OPEN 牌子。当一个人说 "I'm not available"，灯灭了，门锁了——人还在里面，但你进不去。

## 4. 多面体：核心义项展开

### 义项 A：可获得的、可使用的（形容物）

> *Tickets are available online from Monday.*
> 周一起可以在网上买到票。

这是最基础的用法。东西存在 + 你能拿到 = available。和"一句话本质"的映射：门没锁，你伸手就够。

### 义项 B：有空的（形容人）

> *Are you available for a meeting at 3?*
> 你三点有空开会吗？

人也可以是那扇没上锁的门。这个义项和 A 的统一性在于：**不管是物还是人，available 描述的都是"此刻可以被连接"的状态**。一个人 available，意思是你可以触达他的时间和注意力。

### 义项 C：（感情上）单身的/可以交往的

> *So... is she available?*
> 所以……她单身吗？

这个义项在口语中非常活跃，经常带一点暧昧或试探的口气。本质还是那扇门——感情的门没有被别人占据，你有机会。

### 义项 D：可利用的（资源/手段）

> *We must use every available means to find a solution.*
> 我们必须动用一切可以动用的手段来找到解决方案。

这个用法里 available 回到了词源中"有效力的"那层底色。不只是"在那儿"，还暗含"**能派上用场的**"。

## 5. 裂缝标注

### 常见误用

**❌ 把 available 当 "免费" 用**

> ~~The software is available.~~ （想说"免费"）

available 只说"能拿到"，不说"不要钱"。免费用 **free** 或 **free of charge**。
正确：*The software is available for free.*

**❌ available 后面介词搞错**

- available **to** sb — 对某人而言可获得（强调接收方）
- available **for** sth — 可用于某事/某目的（强调用途）
- available **from** — 可从某处获得（强调来源）
- available **in** — 有某种规格/版本（强调形式）

> *This report is available **to** all employees.*（所有员工都能看到）
> *She's available **for** comment.*（可以接受采访）
> *Available **from** all major bookstores.*（各大书店有售）
> *Available **in** three colors.*（有三种颜色可选）

### 易混近义词辨析

| 词 | 区别 |
|---|---|
| **accessible** | 强调"能接近、无障碍"——一栋楼 accessible 是说轮椅能进去，available 是说有房间空着让你住 |
| **obtainable** | 更书面，常暗示需要一些努力才能得到；available 更日常，暗示触手可及 |
| **vacant** | 专门形容物理空间（房间、座位、职位）是空的；available 的范围大得多 |
| **free** | 形容人时，free 侧重"没有被束缚/没在忙"，available 侧重"可以被安排" |

场景对比：
- 酒店房间：*vacant* = 没人住；*available* = 没人住 **且你可以预订**（有些房间 vacant 但在维修，not available）
- 医生：*free* 下午三点 = 他那时候没事做；*available* 下午三点 = 他那时候可以见你（可能他有事，但愿意为你空出来）

### 中文母语者陷阱

**"有" ≠ available。** 中文说"这个有吗？"，英文不能说 "Is this available?" 在所有场合。如果你问店员"有没有这双鞋的 42 码"，英文更自然的说法是 *Do you have this in size 42?* 而不是 *Is size 42 available?*（后者更像在问库存系统）。

**"方便" ≠ available。** 中文说"你什么时候方便？"直译成 *When are you available?* 其实是对的——但要注意，中文的"方便"还带一层礼貌暗示（"我不想打扰你"），available 没有这层客气，它是中性的调度用语。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境 |
|------|------|
| **readily available** | 很容易就能拿到。"Information is readily available online." 比单独 available 多一层"毫不费力"。 |
| **commercially available** | 已经在市场上能买到了（常用于科技/医药——这个药已经 commercially available 了 = 不是实验阶段了）。 |
| **publicly available** | 公开可查的。用于数据、文件、报告。 |
| **make sth available** | 主动让某物变得可获取。"The company made the data available to researchers." 这个搭配暗示之前是不开放的。 |
| **the best available** | 现有条件下最好的。"We used the best available evidence." 带一丝无奈——不是最好的，是能拿到的里面最好的。 |
| **immediately available** | 马上就能到手。 |
| **no longer available** | 不再可得。电商里见到这句话 = 别想了。 |
| **available resources** | 可用资源。正式/商务场合高频。 |

### 语域标注

**全域通用。** available 是英语里少有的横跨所有语域都毫无违和感的词。学术论文里写 *available data*，朋友聊天说 *I'm available Friday*，商务邮件写 *Please let me know your available times*——没有任何场合会因为用了 available 而显得太正式或太随意。

它是真正的"百搭词"——但正因为如此，高级写作中有时会嫌它太平淡，换用更精确的同义词（accessible, at hand, on offer, up for grabs）。

### 活用示范

> *"He made himself emotionally available for the first time in years."*
> 他多年来第一次在情感上敞开了自己。

这里 available 不是在说"有空"，而是在说一个人终于把那扇门打开了——回到我们的便利店画面，他终于把"OPEN"牌子亮了。这种用法在心理学/亲密关系语境中非常有力。

> *"Justice should be available to all, not just to those who can afford it."*
> 正义应该是所有人都能触及的，而不只是那些付得起钱的人。

available 在这里承载了一种政治立场——把抽象概念（正义）当作可以被分配的资源来讨论。

## 7. 跨语言映射

中文里最接近 available 的概念其实不是一个词，而是**两个词的交集**：

- **"有"** ≈ 东西存在
- **"能（拿到/用）"** ≈ 你有获取的能力和通道

available = "有" + "你能拿到"。中文需要两步才能表达的东西，英文一个词搞定。这也是为什么翻译 available 的时候经常会犹豫——到底翻"有"、"可用"、"在"、"有空"还是"可以"？答案是：看你强调的是哪一半。

日语的 **利用可能（りようかのう）** 相当精确地对应了 available 的"可利用"义项，但日语另有 **空いている（あいている）** 来形容人有空或座位空着——available 在英语里一个词覆盖了日语的两个表达。

德语的 **verfügbar** 是最接近的对应词，结构也类似：*ver-*（可以）+ *fügen*（安排）+ *-bar*（能够被）= "能够被安排使用的"。和 available 的语义几乎完全重叠。
