# analysis

---

## 1. 一句话本质

**analysis 是把一团缠在一起的绳结，一根一根抽出来摊平在桌面上的动作——不是为了毁掉绳结，是为了看清每根绳子的走向。**

关键词不是"切碎"，是"松开"。它的原始意象是解绳子，不是砸东西。

---

## 2. 词源考古

### 词根词缀拆解

```
analysis
├── ana-（ἀνά）= 向上、回溯、贯穿
└── -lysis（λύσις）= 松开、解开
     └── lyein（λύειν）= 解开、松绑
         └── PIE *leu- = 松开、切分
```

**ana-** 这个前缀有三重人格："向上"（anatomy：向上切 → 解剖）、"回溯"（anachronism：时间回溯 → 时代错误）、"贯穿/遍及"（analysis 里取的就是这个——"彻底地松开"）。

**-lysis** 是希腊语 λύσις，来自动词 λύειν（lyein），本义就是"解开绑绳"。你认识的一大堆医学和化学词汇都住在这个词根里：

| 词 | 拆法 | 意思 |
|---|---|---|
| **paralysis** | para-（旁边）+ lysis | 神经"松脱了"→ 瘫痪 |
| **dialysis** | dia-（穿过）+ lysis | 穿过膜把东西"分离"→ 透析 |
| **catalysis** | kata-（向下）+ lysis | 把反应"往下松"→ 催化 |
| **electrolysis** | electro + lysis | 用电把物质"拆开"→ 电解 |
| **hydrolysis** | hydro-（水）+ lysis | 用水"拆分"化学键 |

所以 analysis 的字面意思：**彻底地松开（一团东西），把它拆散还原成各个组成部分**。

### 历史流变

**公元前 4 世纪**：亚里士多德和欧几里得已经在用 ἀνάλυσις。在希腊数学里，analysis 是一种证明策略——**从结论反推回已知条件**，和 synthesis（从已知条件正推到结论）构成一对。注意：最早的 analysis 不是"把东西拆碎"，而是"从终点倒着走回起点"。这个"逆向追踪"的味道今天还活着——"分析原因"本质上就是从结果倒推。

**16 世纪 80 年代**：英语正式收编这个词。最初几乎专属数学和逻辑领域。

**17 世纪**：化学家开始说 chemical analysis——把混合物拆解成元素。词义从抽象的逻辑操作落地到具体的物质分离。

**19 世纪末—20 世纪初**：弗洛伊德把 Analyse 塞进了心理学，psychoanalysis（精神分析）诞生。这次语义漂移非常大——从"拆解物质/逻辑命题"跳到了"拆解人的潜意识"。analysis 从此获得了一层"深挖、揭露隐藏结构"的含义。

**21 世纪**：data analysis、business analysis 成了职业头衔。analysis 变成了最泛化的"理性拆解"代名词。

### 文化地层

最值得记住的一层：**analysis 和 synthesis 是一对出生在希腊几何课堂上的双胞胎**。analysis 拆开，synthesis 合上。拆是为了看清，合是为了建造。你今天在学术写作里看到的 "analytical framework" 和 "synthesis of findings"，本质上还是两千四百年前欧几里得的那套逻辑体操。

## 3. 故事 / 画面

不要想实验室。想一个老钟表匠。

他面前是一块怀表——不走了。他没有对着表壳发脾气，也没有摇一摇碰碰运气。他翻过来，用一把细到夸张的螺丝刀，把后盖拧开。然后他开始拆：表壳、表盘、分针、时针、发条、擒纵轮、游丝、宝石轴承……每一个零件被镊子夹出来，按顺序排在绒布上。

他不急。他盯着每个零件看齿痕、看磨损、看弯折。最后他发现——游丝变形了，多出了半圈弧度，导致摆轮走快。

**这就是 analysis**。不是暴力拆解，是有序地松开，是把一个整体还原成零件排列好，然后在零件级别找到问题在哪里。

钟表匠做完 analysis 之后还要把零件装回去——那是 synthesis。但如果没有那个"全摊在绒布上"的步骤，他永远不会知道游丝弯了。

## 4. 多面体：核心义项展开

### 义项一：把整体拆解成部分进行审视（最核心）

> *A detailed analysis of the crash revealed that the failure originated in a single faulty sensor.*
> 对坠机事件的详细分析表明，故障源于一个有缺陷的传感器。

与"一句话本质"的关系：把坠机这个"绳结"一根根抽出来——天气、机械、人为操作、软件——直到找到那根打死结的绳子。

### 义项二：化学分析——确定物质的组成成分

> *The lab ran an analysis of the water sample and found elevated levels of lead.*
> 实验室对水样进行了分析，发现铅含量超标。

关系：字面意义上的"松开"——把水这个混合物拆成各个化学成分，看里面有什么、有多少。这是最接近希腊语原义的一个义项。

### 义项三：精神分析 / 心理分析

> *She's been in analysis for three years and says it changed how she understands her relationships.*
> 她接受精神分析已经三年了，她说这改变了她理解人际关系的方式。

关系：弗洛伊德式的"松开"——把潜意识里缠成一团的欲望、恐惧、创伤一根根抽出来，摊在治疗师和患者之间的桌面上。

### 义项四：in the final/last analysis = 归根结底

> *In the final analysis, it all comes down to whether people trust you.*
> 归根结底，一切取决于人们是否信任你。

关系：当你把所有绳子都抽完、所有零件都摊开看完之后，最后那个底层结论——"终极的那一次松绑之后剩下的东西"。

### 底层统一性

四个义项的共性：**把纠缠的东西松开，逐一审视，找到隐藏在整体之下的结构或真相。** 不管松开的是一个机械故障、一杯水、一个人的潜意识，还是一个复杂论证，动作模式完全一样。

## 5. 裂缝标注

### 常见误用场景

**❌ 把 analysis 当"观点/看法"用**

> 误：*My analysis is that the movie was boring.*

这句话语法没错，但语域不对——你只是觉得电影无聊，这不叫 analysis。analysis 暗示你做了系统的拆解工作（节奏、角色弧光、叙事结构……），不只是一个感受。如果你只是表达感受，用 view / opinion / take。

**❌ 复数形式出错**

analysis 的复数是 **analyses**（读作 /əˈnæl.ɪ.siːz/），不是 ~~analysises~~。这是希腊语复数变化的遗产（-sis → -ses），和 crisis/crises、thesis/theses 一个规律。

### 易混近义词辨析

| 词 | 区别 |
|---|---|
| **analysis** | 系统地拆解，找出组成部分和内在关系。强调**方法论上的严谨性**。 |
| **examination** | 仔细看、检查。不一定要拆解，可以是表面的审视。医生 examine 你的喉咙，不叫 analyze。 |
| **study** | 长时间的、持续的研究。比 analysis 时间跨度更长、范围更宽。"a study of climate change" 可能包含很多次 analysis。 |
| **evaluation** | 评估价值或质量。analysis 找的是"是什么/为什么"，evaluation 给的是"好不好/值不值"。 |
| **review** | 回顾+评价。带有"重新审视"的意味，比 analysis 更轻量、更主观。 |

**关键区分**：analysis 是唯一强调"拆开看"的那个词。其他词都不要求你把整体还原成零件。

### 中文母语者陷阱

**"分析"≈ analysis，但中文里"分析"的使用范围比 analysis 宽得多。**

中文里你可以说"帮我分析一下他是什么意思"——这其实更接近 figure out 或 interpret。英文中 analyze 暗含一套更系统的操作，不是随口"想想"。

另一个陷阱：中文"分析"经常独立做谓语（"你分析得对"），但英文中 analysis 作名词使用的频率远高于 analyze 作动词——学术写作中 "the analysis shows" 比 "we analyzed" 更常见。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境 |
|---|---|
| **conduct / carry out / perform an analysis** | 正式场合的标配三件套。学术论文、商业报告里最常见。 |
| **in-depth analysis** | 深度分析。暗示不是蜻蜓点水，真的钻下去了。 |
| **critical analysis** | 批判性分析。学术写作核心词组，意思是"带着质疑视角去拆解"。 |
| **data analysis** | 数据分析。21 世纪使用频率爆炸式增长的搭配。 |
| **cost-benefit analysis** | 成本效益分析。商业决策标配。 |
| **statistical analysis** | 统计分析。社科和自然科学论文的常客。 |
| **under analysis** | 正在被分析中。比 "being analyzed" 更正式。 |
| **on closer analysis** | 仔细分析之后……。用于引出和初始印象不同的结论。 |

### 语域标注

**偏正式。** analysis 天然带学术气质。日常口语中人们更常说 "let me think about it" 或 "let's break it down" 而不是 "let me do an analysis"。口语中如果有人说 "my analysis of the situation"，要么是在半开玩笑地正经，要么是在正式讨论场景中。

在新闻语境中非常高频——"analysis" 是新闻标题的常客（"Analysis: Why the ceasefire failed"），这里 analysis 几乎变成了一种文体标签，意思是"这篇文章不是报道事实，是解读事实"。

### 活用示范

> *Enough analysis. Ship it.*
> 别分析了。上线。

——硅谷式反讽。analysis paralysis（分析瘫痪）的精神延伸：分析到最后变成了逃避行动的借口。这个用法把 analysis 从褒义词（理性、严谨）翻转成了贬义（拖延、怯懦）。

> *His songs don't survive analysis — they survive feeling.*
> 他的歌经不起分析——但经得起感受。

——这句话把 analysis 放在"感受"的对立面，暗示 analysis 是一种冷的、解剖式的操作，会杀死某些只能整体感受的东西。非常精准的活用。

## 7. 跨语言映射

**中文"分析"**：字面意思"分开"+"析出"（析的古义是劈木头），和 analysis 的"松开"异曲同工——都是用物理性的"拆分"动作隐喻智力操作。两个文明独立发明了几乎一样的隐喻，说明这个认知模式是人类共通的。

**日语「分析」（ぶんせき）**：直接从中文借来的汉字词，用法和中文几乎一致。但日语口语中更常用「見てみる」（看看）来代替正式的「分析する」，和英语口语中 analysis 偏正式的情况类似。

**德语 Analyse**：几乎是英语的同源词，但德语中 Analyse 比英语的 analysis 在日常对话中出现频率更高——德国人说话本来就更"学术味"一些。

**拉丁语 → 值得注意的反向词**：analysis 的反义词 **synthesis** 也是希腊语双胞胎（syn- = 一起 + thesis = 放置 → 放到一起）。在德国古典哲学（尤其是黑格尔）里，Analyse 和 Synthese 构成了辩证法的两个基本动作——正题（thesis）、反题（antithesis）、合题（synthesis），而 analysis 是把合题再拆回去看的那个力。

---

**记忆锚点回检**：闭上眼——一个老钟表匠，把怀表的零件一个个镊出来，排在绒布上。analysis 就是那块绒布上整齐排列的零件。有这个画面，词义不会忘。
