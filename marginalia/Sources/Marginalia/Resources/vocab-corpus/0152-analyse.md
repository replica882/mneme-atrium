# analyse

---

## 1. 一句话本质

**analyse 是把一团打了死结的毛线，耐心地、一根一根地拆开，直到看清每根线从哪来、往哪去。**

不是剪断，不是撕开——是"解"。手指捏住一个线头，顺着纹路，慢慢松开结构。拆完之后毛线还在，但你终于看懂了它是怎么缠在一起的。

---

## 2. 词源考古

### 词根词缀拆解

```
ana-  +  lyse / lysis
↑          ↑
向上/向回    松开、解开
(希腊语)    (希腊语 luein λύειν)
```

- **ana-**（ἀνά）：希腊语前缀，核心义"向上、向回、贯穿"。这里取"throughout / 彻底地"的意味——不是随便拆一下，是**从头到尾地**解开。
- **lysis / lyse**（λύσις / λύειν）：希腊语"松开、释放、溶解"。这个词根是个大家族——paralysis（瘫痪 = 在旁边松掉了，肌肉失控）、catalyst（催化剂 = 往下松解）、dialysis（透析 = 穿过地分离）。

所以 analyse 的原始画面是：**把一个东西彻底地松开**。

### 历史流变

| 时间 | 事件 |
|------|------|
| 古希腊 | ἀνάλυσις（analusis）最初用在几何学里——把一个复杂命题**倒推回**已知公理。亚里士多德和欧几里得都在用。注意：这时候的"分析"是**逆向推理**，不是我们今天说的"拆分"。 |
| 中世纪拉丁语 | analysis 进入学术拉丁语，保持"逆推法"的含义。 |
| 16世纪 | 经法语 analyser 进入英语。最初只在化学和逻辑学里用——化学家"analyse"一种物质 = 把它分解成组成元素。 |
| 17-18世纪 | 含义开始泛化。从"分解物质"扩展到"分解论证、分解问题、分解文本"。 |
| 19世纪末 | 弗洛伊德把这个词带进了心理学。psycho-analysis = 把心理结构拆开来看。这一下子让 analyse 带上了"深挖内心"的文化色彩。 |
| 现代 | 全面泛化——分析数据、分析市场、分析比赛、分析关系……几乎任何"仔细看看这是怎么回事"都可以用。 |

### 拼写分裂：analyse vs. analyze

这不是对错问题，是大西洋两岸的分家：

- **analyse**（-yse）：英式拼法，保留了法语 analyser 的尾巴。英国、澳大利亚、新西兰用这个。
- **analyze**（-yze）：美式拼法。美国人在 18-19 世纪系统性地把一批 -yse 词改成了 -yze（类比 -ize 的拼写传统）。

两者发音完全一样，含义完全一样。选哪个纯粹看你要跟哪边的英语保持一致。

### 文化地层：从"逆推"到"拆解"

最值得记住的转折：古希腊人说 analysis，脑子里想的是"从结论倒着走回前提"——是一条**路径**。现代人说 analyse，脑子里想的是"把东西拆成零件"——是一次**手术**。

从"走回去"到"拆开来"，这个意象漂移花了两千年。但底层没变：**让隐藏的结构变得可见**。

## 3. 故事 / 画面

一个钟表匠坐在灯下。

面前是一块怀表，走不准了。他没有摇一摇、拍两下、猜一个原因。他拿出工具，卸下后盖，取出摆轮，拆下游丝，把每一个齿轮都摊在绒布上，按顺序排好。

他不修。他先只做一件事：**看**。

每个零件单独审视——这个齿轮有磨损吗？那根弹簧弹性还在吗？这两个咬合面之间的间隙是不是比出厂时大了？

等他看完所有零件、搞清了每一层关系之后，问题的答案自己就浮出来了——原来是第三级齿轮的轴承偏了 0.2 毫米。

**这就是 analyse。不是修，不是猜，不是下结论。是拆开、摊平、逐一审视，让结构自己说话。**

（而"修"那一步，叫 solve。analyse 只负责让你看清楚。）

## 4. 多面体：核心义项展开

### 义项 A：分解研究（最常用）

> *We need to **analyse** the survey results before making any decisions.*
> 我们得先分析调查结果再做决定。

和本质的关系：把一团数据"松开"，看看里面的结构。

### 义项 B：化学/科学分解

> *The lab will **analyse** the water sample for heavy metals.*
> 实验室会分析水样中的重金属含量。

这是最接近词源原始画面的用法——字面意义上"把物质拆解成元素"。

### 义项 C：心理分析

> *After years of being **analysed**, she finally understood her fear of abandonment.*
> 经过多年的心理分析，她终于理解了自己对被抛弃的恐惧。

弗洛伊德版的"拆开"：把潜意识里打结的东西松解开。

### 义项 D：语法/文本分析

> ***Analyse** the sentence structure and identify the subordinate clauses.*
> 分析句子结构，找出从句。

把一个完整的句子"拆"成主谓宾定状补——和钟表匠拆齿轮是同一个动作。

### 底层统一性

四个义项，同一个内核：**把一个整体拆成可辨识的部分，以便理解其结构**。无论拆的是数据、物质、心理还是句子，动作的本质没变过——从希腊几何学家到今天的数据科学家，干的是同一件事。

## 5. 裂缝标注

### 常见误用

**❌ analyse ≠ think about**

很多人把"I need to analyse this"当成"I need to think about this"的高级说法。不是。analyse 暗示了一个**系统性拆解**的过程。如果你只是在琢磨要不要换工作，那是 think about / consider，不是 analyse。

> ✗ *I analysed whether to go to the party.* （太重了——你不是在做系统性拆解）
> ✓ *I considered whether to go to the party.*

**❌ analyse ≠ describe**

"分析"和"描述"在中文里经常混用，英文里它们是两件事。Describe 是"说它长什么样"，analyse 是"说它为什么长这样"。

> ✗ *Analyse the painting* → 然后只说"画面上有一棵树和一个人"（这是 describe）
> ✓ *Analyse the painting* → "画家用冷色调和不对称构图制造了一种疏离感"（这是 analyse）

### 易混近义词辨析

| 词 | 区别 |
|------|------|
| **examine** | 仔细看，但不一定拆开。医生 examine 你 = 检查；analyse 你的血样 = 把血分解成指标。examine 是用眼睛看，analyse 是用手术刀拆。 |
| **evaluate** | 方向不同。analyse 是"这东西由什么组成"，evaluate 是"这东西值不值"。analyse 是拆，evaluate 是称重打分。 |
| **assess** | 和 evaluate 类似，但更偏"估量风险/情况"。assess the situation = 估一下局势如何。analyse the situation = 拆解这个局势的各个因素。 |
| **study** | 更宽泛。study 可以是漫长的、开放式的探索；analyse 暗示一个有方法论的、结构化的拆解过程。 |

### 中文母语者陷阱

**"分析"在中文里太好用了。**

中文的"分析"几乎可以用在任何"动脑子"的场景——"分析一下为什么他生气了""你分析分析该不该买"。但英文的 analyse 比中文的"分析"**重**得多。它暗示的是一个**有方法、有步骤、有深度的拆解过程**，不是随口聊聊。

日常口语里"你帮我分析分析"，英文更可能用 *figure out*、*think through*、*look into*，而不是 analyse。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境 |
|------|------|
| **analyse data / results** | 数据分析——办公室里最高频的用法。 |
| **analyse the situation** | 拆解局势，通常用在需要冷静判断的场合。 |
| **carefully / thoroughly analyse** | 强调拆得仔细、拆得彻底。 |
| **critically analyse** | 学术写作标配——"批判性分析"，意思是不只拆，还要评判每个部件的可靠性。 |
| **analyse a problem** | 不是"想想这个问题"，是"把问题拆成可处理的小块"。 |
| **analyse a text / poem / speech** | 文学/语言学分析——拆结构、拆修辞、拆意图。 |
| **analyse trends / patterns** | 商业/社科——从数据里拆出趋势和规律。 |

### 语域标注

**偏正式。** analyse 不是一个日常闲聊词。你不会对朋友说"let me analyse why you're upset"（这会显得像心理咨询师上线了）。它的主场是：

- 📄 学术写作（最自然的栖息地）
- 📊 商业报告
- 🔬 科学研究
- 📰 严肃新闻评论
- 💬 正式讨论

日常口语里，*look at*、*break down*、*figure out* 往往是更自然的选择。

### 活用示范

> *"Don't just feel your anger — **analyse** it. Where does it live in your body? What triggered it? What story is it telling?"*
> "别光生气——**拆解**它。它住在你身体的哪里？什么触发了它？它在讲什么故事？"

这句把 analyse 用在情绪上，效果特别好——因为它暗示：你的情绪不是一团浑浊的东西，它有结构、有来源、有零件，你可以一个一个地拆开看。

> *"The best coaches don't just watch the game — they **analyse** the space between players."*
> "最好的教练不只是看比赛——他们**分析**的是球员之间的空间。"

这句展示了 analyse 最优雅的用法：不是分析显而易见的东西（球员的跑位），而是分析**看不见的结构**（空间关系）。

## 7. 跨语言映射

**中文"分析"**：覆盖面比 analyse 宽得多。中文的"分析"可以很轻（"你分析分析"≈ 你说说看），也可以很重（"深入分析"）。英文的 analyse 几乎总是在重的那一端。

**日文「分析する」（ぶんせきする）**：和英文 analyse 的正式程度几乎完全对齐——都是书面/学术/严肃语境的词。日常说"看看怎么回事"也不会用「分析する」。

**德文 analysieren**：直接借自法语/拉丁语，用法和英文高度重合。有意思的是德语还有一个日耳曼本土词 **zergliedern**（zer- = 拆散 + Glied = 肢体/环节），字面意思是"把肢体拆开"——比 analysieren 更暴力、更具象，只在学术语境用。

**法文 analyser**：英文 analyse 的直系亲本。法语里这个词的日常化程度比英语高一些——法国人可以比较随意地说 *analyse la situation*，不像英语那么端着。
