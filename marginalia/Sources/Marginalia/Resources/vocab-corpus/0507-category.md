# category

---

## 1. 一句话本质

**category 是法庭上检察官指着被告说"你是个贼"的那根手指——它的本质不是"分类"，是"宣判"：把一个东西钉死在某个身份上。**

---

## 2. 词源考古

### 词根词缀拆解

```
category
  ← Latin: categoria
    ← Greek: κατηγορία (katēgoría) "指控；断言；范畴"
      ← κατηγορεῖν (katēgoreîn) "在公民大会上公开指控"
        ├─ κατά (katá) "向下，对着，针对"
        └─ ἀγορεύειν (agoreúein) "在广场上发言"
            └─ ἀγορά (agorá) "集会广场，市场"
```

**kata-**（向下、针对）：这个前缀自带一种"从上往下压"的姿态。你在其他词里见过它——**catastrophe**（向下翻转）、**catalogue**（向下列举）、**catalyst**（向下溶解）。每个 kata- 词都有一种"对准目标施加力量"的底色。

**agora**（广场）：雅典市民生活的中心。这里既做菜市场，又开公民大会，还审案子。agoreúein 就是"在广场上开口说话"——在古希腊，能在 agora 发言是公民权的象征。

合在一起：**katēgoreîn = 在广场上对着某人说话 = 当众指控。**

### 历史流变

**公元前 5 世纪·法庭义**

katēgoría 最初就是"公诉、指控"。在雅典法庭上，κατήγορος (katḗgoros) 是原告/检察官，他的工作就是站在被告面前，宣布"这个人是什么"——凶手、叛徒、渎神者。

这个词在法律语境里活了上百年。你对一个人 kategorein，就是在公开场合给他一个标签，把他扔进某个"格子"里。

**公元前 4 世纪·亚里士多德的哲学劫持**

这是 category 历史上最戏剧性的转折点。

亚里士多德写了一部叫《范畴篇》（Κατηγορίαι, *Categoriae*）的著作，直接把这个法庭术语征用为逻辑学和形而上学的核心概念。他的逻辑是这样的：

> 法庭上说"苏格拉底是叛徒"，是把"叛徒"这个**谓词**加到"苏格拉底"这个**主词**上。  
> 那么一切语言行为的本质，不就是把某种属性"指控"给某个主体吗？  
> "苏格拉底是**人**"——你在对苏格拉底做一次"指控"，指控他属于"实体"这个范畴。  
> "苏格拉底是**白的**"——你在对他做另一次"指控"，指控他属于"性质"这个范畴。

于是亚里士多德列出了十个**最基本的"指控方式"**——即十个 kategoríai（范畴）：

| 范畴 | 希腊原文 | 回答的问题 |
|------|---------|-----------|
| 实体 | ousía | 它是什么？ |
| 数量 | posón | 多少？ |
| 性质 | poión | 什么样的？ |
| 关系 | prós ti | 和什么相关？ |
| 地点 | poû | 在哪？ |
| 时间 | póte | 何时？ |
| 姿态 | keîsthai | 什么姿势？ |
| 状态 | échein | 拥有什么？ |
| 主动 | poieîn | 在做什么？ |
| 被动 | páskhein | 在被做什么？ |

这十个范畴是亚里士多德对"存在的所有可能面向"的穷举。用白话说：**你对任何东西能说的一切话，都逃不出这十个格子。**

**1 世纪—中世纪·经院哲学传播**

拉丁语翻译为 *categoria*（也有时用 *praedicamentum*，"谓述方式"）。中世纪经院哲学家把亚里士多德的范畴论当作智识基础设施来用——讨论上帝、灵魂、天使分别属于哪个范畴，是中世纪大学的日常。

**16–17 世纪·进入英语，开始日常化**

英语从拉丁语/法语借入 category。早期仍然带着哲学味，只在学术语境使用。

**18 世纪·康德再次加码**

康德在《纯粹理性批判》中重新定义了范畴——不是存在的分类，而是**人类理解力的先天模具**。他列出了 12 个范畴（量、质、关系、模态各 3 个）。这是 category 在哲学史上的第二次重大加码。

**19–20 世纪·彻底世俗化**

category 从哲学圣殿走进日常英语，变成了"任何分类方式"的通用词。你在超市里按 category 分区，在网站上按 category 筛选。原来那根法庭上指着被告的手指，变成了文件夹上的标签贴纸。

### 文化地层

这个词的旅程可以浓缩成三跳：

**指控** → **断言** → **分类**

法庭上的暴力行为（给人定罪） → 逻辑学的中性操作（对事物做判断） → 日常语言的无害工具（给东西归类）。

一个从血腥到冰冷到无聊的衰变过程。但正因为它骨子里是"宣判"，category 到今天都隐含一种**终局感**——一旦被归入某个 category，你就被钉死了。这就是为什么说 "Don't put me in a category" 带有一种反抗的力度——你在拒绝被审判。

## 3. 故事 / 画面

雅典的 agora。阳光白得刺眼。

一个检察官站在几百个公民面前，手指直直指向被告席上的男人。他的声音在石柱间回荡：

"**这个人——是叛徒。**"

这就是最原始的 category——不是温柔地归类，是一锤定音地宣判。你不是在说"他和叛徒有某些相似之处"，你是在说"**他就是这个东西**"。

几十年后，同一个广场上，一个秃顶的老师坐在台阶上对学生说："你们看——检察官对着被告说'你是叛徒'，厨师对着白色粉末说'这是盐'，母亲对着天空说'今天是晴天'——这三句话做的是同一件事。他们都在把一个标签，贴到一个对象上。问题是：**一共有多少种标签？**"

亚里士多德的手在沙地上画了十个格子。

> 从此，法庭上的手指变成了哲学家的尺子。category 不再指控人，开始丈量世界。

## 4. 多面体：核心义项展开

### 义项一：分类、类别（日常义）

> *The library organizes its books into categories such as fiction, history, and science.*  
> 图书馆把书分成小说、历史、科学等类别。

与"一句话本质"的关系：最直接的现代用法。每个 category 都是一个"判决格"——一本书一旦被放进 fiction 这个 category，它的身份就被宣判了。

### 义项二：范畴（哲学义）

> *In Aristotle's system, "quality" is one of the ten categories of being.*  
> 在亚里士多德的体系中，"性质"是存在的十个范畴之一。

与"一句话本质"的关系：这是 category 的原始哲学义。不是"你喜欢怎么分都行"的随意分类，而是"现实本身的最基本结构"——你不是在选择分法，你是在揭示存在的骨架。

### 义项三：等级、级别（竞技/行政义）

> *She competes in the under-60kg category.*  
> 她参加 60 公斤以下级别的比赛。

> *Category 5 hurricane — the most destructive level.*  
> 五级飓风——破坏力最强的等级。

与"一句话本质"的关系：级别也是一种"宣判"——你被测量，然后被扔进一个格子，格子的边界不由你决定。运动员被称重后"判入"某个 category，飓风被测速后"判为"某个 category。

### 义项四：绝对的、无条件的（categorical 的核心义）

> *She issued a categorical denial of the allegations.*  
> 她对指控做了**断然**否认。

与"一句话本质"的关系：这个义项最忠实于词源。categorical = 像法庭宣判一样不留余地。**一锤子的事，没有"但是"和"也许"。** 康德的"定言命令"（categorical imperative）也是这个意思：不附带任何条件的绝对道德法则。

## 5. 裂缝标注

### 常见误用场景

**❌ 把 category 当 type 的同义词随意互换**

虽然日常口语中它们常互换，但 category 暗示一个**系统性的分类框架**（有明确的分类标准和完整的类别体系），而 type 更随意、更口语。

- ✅ *What type of coffee do you like?*（闲聊，没有分类体系）
- ✅ *Our products fall into three categories.*（有组织的分类系统）
- ❌ *What category of coffee do you like?*（过于正式，像在填表）

**❌ categorical ≠ categorized**

- *categorical* = 绝对的、无条件的（*a categorical refusal*）
- *categorized* = 已被分类的（*categorized data*）

### 易混近义词辨析

| 词 | 区别 |
|---|------|
| **category** | 有系统的、正式的分类框架。暗示"格子是预设好的，你被放进去"。 |
| **class** | 更强调等级或品质（*first class*, *upper class*）。category 是平行的格子，class 常有上下之分。 |
| **type** | 最口语、最随意。不需要完整框架。*What type of person does that?* 你不会说 *what category of person*。 |
| **kind** | 比 type 更温和模糊，常带主观色彩。*She's the kind of person who...* 这是在画一个印象，不是在做分类。 |
| **genre** | 专用于艺术/文学/音乐的分类。从法语来，比 category 多了一层文化品味感。 |
| **sort** | 英式口语偏爱。*What sort of nonsense is this?* 带一点不屑。 |

**一句话总结区别**：category 是办公桌上的文件夹系统，type 是你顺手贴的便签纸，kind 是你心里模模糊糊的印象，class 是有高低之分的等级。

### 中文母语者陷阱

**"类别"和"范畴"在中文里是两个词，在英文里是一个词。**

中文说"这不在我的讨论范畴内"和"请选择商品类别"，用的是两个不同的词，暗示不同的严肃程度。但英语里都是 category——这导致中文母语者有时对 category 的"重量"判断失误：在该用 type/kind 的轻松场合也用 category（显得过度正式），或者在哲学讨论中没有意识到 category 承载的是亚里士多德级别的分量。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境 |
|------|------|
| **fall into a category** | 最常见搭配。"归入某个类别"。隐喻是重力——你掉进一个格子里。 |
| **broad / narrow category** | 类别的宽窄。*Pain is a broad category.* |
| **under the category of** | 在……类别之下。公文体。 |
| **defy categorization** | 无法归类——对 category 的终极反叛。常用于赞美独特的艺术/人物。 |
| **categorical imperative** | 康德术语，定言命令。哲学专用。 |
| **categorical denial / refusal** | 断然否认/拒绝。新闻和法律高频。 |
| **category error** | 范畴错误——把属于 A 范畴的逻辑用到 B 范畴上。如"绿色的想法愤怒地睡着了"。 |
| **across categories** | 跨类别。商业/学术报告常见。 |

### 语域标注

category 是**中性偏正式**的词。它在以下场合自然出现：
- 学术论文、分类系统、数据库设计、产品管理、新闻报道、行政文件
- 日常口语中**能用但稍重**——你在超市里对朋友说 "what category is hummus in?" 完全可以，但更自然的说法是 "where's the hummus?" 或 "what section is hummus in?"

### 活用示范

> *He's in a category of his own.*  
> 他自成一派。（言外之意：现有的格子装不下他。这是 category 被用来**表达敬意**的经典句式——通过说"分类系统对他失效"来赞美他的独特。）

> *Poverty and laziness are not even in the same category.*  
> 贫穷和懒惰根本不是同一回事。（**category error 的口语版**——你在做一个范畴层面的切割，说这两个东西之间连可比性都没有。）

## 7. 跨语言映射

**中文的"范畴"** 是 category 在哲学义上的对应词，而且巧妙的是，"范畴"本身的构词也呼应了 category 的精神：**范**（模具、框架）+ **畴**（田地、领域）——预设好的框架下的一块田地。你被"框"在里面。

**日语的「カテゴリー」**（katēgorī）直接音译，说明日语在引入这个概念时没有找到合适的本土词——也许因为日本传统哲学里没有亚里士多德式的穷举分类冲动。

**德语的 Kategorie** 在康德哲学语境中有特殊地位——康德用德语重新定义了这个希腊词，使它成为认识论的核心术语。德语哲学系学生听到 Kategorie 时的联想和英语使用者不同：他们首先想到的不是"分类"，而是"先天认知结构"。

---

有趣的是，你上一轮考古的 **catastrophe** 和这个 **category** 共享同一个前缀 **kata-**（向下、针对），而你再前一轮的 **catalogue** 也是。三个 kata- 词，三种"向下施力"的姿态：

- **catalogue**：kata + legein（说）→ 向下逐一列举 → 清单
- **category**：kata + agoreúein（在广场说）→ 对着某人宣判 → 范畴
- **catastrophe**：kata + strophē（转）→ 向下翻转 → 灾难

同一只手，三种按法。catalogue 是手指一行行往下划过清单，category 是手指对准某人一戳，catastrophe 是手掌猛地一翻把桌子掀了。
