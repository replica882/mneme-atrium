# antiquated

---

## 1. 一句话本质

**antiquated 是博物馆橱窗里那台打字机——没人怀疑它曾经好用，但你不会拿它来写今天的邮件。** 它不是"老"，是"老到跟不上了"。

---

## 2. 词源考古

### 拆骨骼

```
anti·qu·ated
│      │   │
│      │   └─ -ated：英语过去分词后缀，表"被……处理过的"
│      └──── -qu-：拉丁语词干残留（antiquus 的尾音）
└─────────── anti-/ante-：拉丁语 ante，"在……之前"
```

核心零件只有一个：**ante（在前面、在之前）**。

拉丁语 **antiquus**（古老的）= ante + 后缀 -icus，字面意思就是"在前面的、先于当下的"。这跟中文"前人"的造词逻辑一模一样——时间线上的"前"就是"古"。

### 历史流变

| 时期 | 发生了什么 |
|---|---|
| 拉丁语 | **antiquāre** = 动词，"宣布作废、废止（一项法律）"。罗马元老院可以 antiquare 一条法案——投票让它"变成旧的"，即否决。这是个**官方行为**，不是自然衰老。 |
| 16世纪晚期 | 英语借入 **antiquate**（动词），意思保留了拉丁语的"使过时"。同期 **antiquated** 作为形容词开始独立使用。 |
| 17–18世纪 | 关键漂移：从"被人为废止"滑向"因为时间流逝而自然过时"。不再需要元老院投票——时间本身就是那张否决票。 |
| 现代英语 | 完全固化为形容词，动词 antiquate 几乎没人用了。含义稳定在"老旧到不实用"。 |

### 文化地层：一个关键分裂

**antiquated** 和 **antique** 明明来自同一个拉丁词根，却走向了截然相反的情感色彩：

- **antique**（名词/形容词）：老，且**因为老而珍贵**。人们花大价钱买 antique furniture。
- **antiquated**（形容词）：老，且**因为老而碍事**。没人夸你的系统 antiquated。

同源词，一个升值一个贬值。分野大约在 18 世纪固化：antique 被收藏家和审美话语征用，antiquated 被效率话语征用。你可以把它想成同一个老人——挂在墙上是 antique，坐在办公桌前挡路就是 antiquated。

## 3. 故事 / 画面

一家公司的服务器机房。角落里有一台米黄色的机器，上面积了灰，风扇还在吱吱转。所有人都知道它还活着，但没人敢碰——因为没人记得它跑着什么程序。新来的工程师指着它问："这是什么？"老员工耸肩："别碰，上次有人拔了它的线，三楼的打印机全瘫了。"

它没坏。它甚至还在工作。但它属于一个已经不存在的时代，靠着惯性苟在现实里。每个人看到它都会想：**该换了，但谁来换？**

这就是 antiquated 的质感——不是废墟的庄严，不是古董的温润，而是**一种尴尬的滞后**。它还在，但它不该还在。

## 4. 多面体：核心义项展开

antiquated 的义域不算宽，但它的"老旧"有几种不同的施力方向：

### 义项 A：技术/系统过时

最高频的用法。指设备、技术、基础设施因为跟不上发展而显得落伍。

> *The factory still relies on antiquated machinery that breaks down every other week.*
> 这家工厂还在用老掉牙的机器，隔三差五就坏。

和本质的关系：打字机式的过时——曾经是标配，现在是累赘。

### 义项 B：观念/制度过时

指法律、规则、思维方式不再适用于当下社会。这个义项保留了拉丁语原始的"法律废止"色彩。

> *Their antiquated hiring practices discourage qualified candidates from even applying.*
> 他们那套陈腐的招聘流程，让有能力的人根本不想投简历。

和本质的关系：不是"传统"，是"该更新但没更新"。重点在**该换了**。

### 义项 C：风格/外表陈旧

较轻的用法，有时近乎打趣，形容某人的穿着、说话方式"像上个世纪的人"。

> *He showed up in an antiquated three-piece suit that even his grandfather would have retired.*
> 他穿了一套老古董三件套，连他爷爷都不会再穿的那种。

和本质的关系：视觉上的时间错位——你看见它就知道它不属于这里。

**底层统一性**：三个义项共享一个结构——**某样东西跨越了它的有效期，还留在现场。** 不是坏了，是过期了。

## 5. 裂缝标注

### 常见误用

**❌ 用 antiquated 形容自然物或纯粹的"年代久远"**

> ~~antiquated mountains~~ ~~antiquated fossils~~

antiquated 只能修饰**人造的、可更新的东西**——系统、制度、观念、技术。山不会过时，因为山不需要升级。化石不会 antiquated，因为化石的价值恰恰在于它老。用 **ancient** 就对了。

**❌ 当褒义词用**

> ~~I love this antiquated little bookshop.~~（语义冲突：你说你爱它，又说它过时碍事？）

如果你想表达"老的、有韵味的"，用 **quaint**、**old-world**、**vintage**。antiquated 自带"该淘汰了"的判断，往好了说也只是中性偏负。

### 易混近义词辨析

| 词 | 核心区别 | 场景测试 |
|---|---|---|
| **antique** | 老且珍贵。收藏家的词。 | 一把 200 年的椅子：antique chair ✓ / antiquated chair ✗（除非你要吐槽它不能坐） |
| **archaic** | 属于远古时期，常指语言或制度。中性偏学术。 | archaic language（古语），antiquated language（过时的措辞）——archaic 的"古"更远、更庄重 |
| **obsolete** | 已经彻底退出使用。比 antiquated 更"死"。 | 传真机：antiquated（还有人用，但很惨）→ obsolete（没人用了） |
| **outdated** | 最日常的"过时"，语气比 antiquated 轻。 | outdated software（该更新了）vs. antiquated software（老到离谱了） |
| **old-fashioned** | 不赶时髦，但可能是主动选择，可以带温情。 | old-fashioned values 可以是褒义；antiquated values 几乎一定是批评 |

### 中文母语者陷阱

**"古老" ≠ antiquated。** 中文的"古老"是中性偏褒的（古老的文明、古老的传说），但 antiquated 偏贬。中文里真正对应 antiquated 语感的是**"老掉牙"**或**"陈旧过时"**——带着"该换了"的不耐烦。

如果你想说"古老而令人敬畏"，用 ancient 或 age-old，千万别用 antiquated。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境 |
|---|---|
| **antiquated system** | 最经典搭配。写改革类议论文必出现。 |
| **antiquated law(s)** | 指早该修订的法律。新闻报道高频。 |
| **antiquated equipment / machinery** | 工厂、医院、学校——设备太旧跟不上。 |
| **antiquated notion / belief** | 指落后的观念。"女性不该工作"之类的就适合这个搭配。 |
| **antiquated infrastructure** | 道路、电网、管道系统年久失修。政策讨论常见。 |
| **seem / look / feel antiquated** | 主观判断——在某人眼里显得过时。 |
| **hopelessly antiquated** | 加强语气：过时到没救了。 |

### 语域标注

**书面 > 口语。** antiquated 是个"写作词"——你在报纸社论、学术论文、科技评论里常见它，日常对话里更可能说 outdated 或 old-school。

正式程度：**中高。** 不至于像 archaic 那么学术，但比 old-fashioned 正式得多。

### 活用示范

> *Silicon Valley loves to call everything antiquated — until the antiquated thing is the only one still working during an outage.*
> 硅谷最爱说什么都过时了——直到断电的时候，过时的那个是唯一还能用的。

这句话利用了 antiquated 的贬义来制造反讽：被嫌弃的"过时"反而成了可靠。

> *The law was so antiquated that it technically made it illegal to carry an ice cream cone in your back pocket on Sundays.*
> 这条法律老旧到什么程度呢——按它的规定，周日在后裤兜里放冰淇淋甜筒是违法的。

美国各州确实有这类古董法律，antiquated 在这种语境里几乎带着喜剧效果。

## 7. 跨语言映射

**中文"陈旧"**是最接近的对应，但"陈旧"更平、更客观。antiquated 多了一层**"早该换了"的催促感**——不只是描述状态，而是在下判断。

**日语「時代遅れ」（じだいおくれ）**覆盖了几乎相同的语义区间——"跟不上时代"，同样偏贬义，同样用于制度、技术、观念。

**法语 suranné** 很有意思：sur-（超过）+ an（年），字面意思是"超过了年限的"。比 antiquated 多一层诗意的惆怅——法国人连吐槽过时都带点忧伤美感。

**拉丁语 obsoletus**（被遗忘的、磨损的）是 antiquated 在罗马时代的近义词，后来英语把它借成了 obsolete，但意思比 antiquated 更绝——obsolete 是死了，antiquated 是还没死但该死了。

---

**记忆锚点回收：** 服务器机房角落那台米黄色的机器。没坏，还在转，但所有人看它的眼神都是同一个意思——*你怎么还在这儿？* 这就是 antiquated。
