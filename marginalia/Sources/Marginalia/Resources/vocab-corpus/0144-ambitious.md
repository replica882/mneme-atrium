# ambitious

---

## 1. 一句话本质

ambitious 是一个人绕着城墙转圈的脚步声——你还没进去，但你已经在丈量它的周长了。不是"想要"，是"围着想要的东西打转，停不下来"。

---

## 2. 词源考古

### 词根词缀拆解

```
amb-（around，四周） + -ire（to go，走） → ambitio（走来走去）→ ambitiosus → ambitious
```

- **amb-**：拉丁语前缀，"围绕、四周"。同族词：ambiance（环绕的氛围）、ambulance（走来走去的→移动医疗车）、ambiguous（两边都能走→歧义的）。
- **-ire / -itus**：拉丁语"走"。它的过去分词 *ambitus* 就是"绑着走了一圈"。
- **-osus**（→英语 -ous）：形容词后缀，"充满……的"。

所以 ambitiosus，字面意思是"充满了绕圈走动的"。

### 历史流变：从骂人话到夸人话

这个词的前世今生是一出完整的翻身剧：

**古罗马（公元前）**：*ambitio* 的原始场景极其具体——罗马政客为了拉选票，穿着白色 toga，在城里挨家挨户"走动"（canvassing），握手、许诺、讨好。所以 ambitio 一开始不是"雄心"，而是**拉票、钻营、跑官**。带着浓重的贬义——你想当官想疯了，所以满街跑。

**中世纪拉丁语 → 中古法语（13-14世纪）**：进入基督教文化圈后，ambitious 的贬义更重了。教会话语体系里，ambition 是**七宗罪的近亲**——对世俗权力的贪欲。但丁《神曲》里的野心家被丢进地狱。中世纪说一个人 ambitious，基本等于说他"不安分、僭越、想爬到不该去的位置"。

**文艺复兴到启蒙运动（16-18世纪）**：关键转折来了。个人主义兴起，"想往上走"不再是道德缺陷，变成了被赞赏的品质。莎士比亚笔下，ambitious 还在褒贬之间摇摆——凯撒的 ambition 是他被刺杀的理由（"As Caesar was ambitious, I slew him"），但这话出自布鲁图斯之口，莎翁让观众自己判断。

**现代英语（19世纪至今）**：ambitious 终于彻底洗白，变成一个以褒义为主的词。"She's very ambitious"在今天的英语里几乎是纯粹的称赞。但——注意——那个古老的阴影没有完全消失。在某些语境下（尤其是英式英语或政治评论中），ambitious 依然可以暗示"太想要了、不择手段"。

### 文化地层

一个词从"跑官钻营"变成"值得尊敬的进取心"，折射的是整个西方文明对**个体欲望**态度的翻转：从基督教的禁欲主义到资本主义的进取伦理。ambitious 的词义漂移，就是这条历史大河的微缩模型。

## 3. 故事 / 画面

罗马城，选举季的清晨。

一个年轻人穿着白得刺眼的 toga candida（候选人穿的白袍，candidate 这个词也从这来），天没亮就出门了。他不是在散步。他在**绕圈**——绕着城区一条街一条街地走，敲门，握手，记住每一个铁匠的名字和每一个寡妇的丧事。他的凉鞋底磨得薄如纸，但他停不下来。因为他脑子里有一个画面：元老院的席位，那把属于他的椅子。

他不是在走路。他是在**丈量自己和那把椅子之间的距离**——然后用脚步一寸一寸地缩短它。

这就是 ambitious 的原始画面：**围着你想要的东西打转的那种不安分**。不是坐着幻想，是已经在行动了，已经在绕圈了，鞋底已经磨穿了。

两千年后，这个动作没变。只不过白袍换成了西装，城区换成了赛道。

## 4. 多面体：核心义项展开

### 义项一：（人）有雄心的、有抱负的

最常用的那张脸。指一个人有强烈的成功欲望，并且愿意为此付出努力。

> *She's one of the most ambitious young directors in the industry.*
> 她是业内最有野心的年轻导演之一。

和"一句话本质"的关系：她在绕着那把导演椅转圈，停不下来。

### 义项二：（计划/项目）宏大的、难度高的

这个义项让很多中文母语者意外——ambitious 不只形容人，还形容**事**。一个 ambitious plan 不是"有野心的计划"，而是"规模大、难度高、需要很大努力才能实现的计划"。

> *The city unveiled an ambitious plan to build a new transit system within five years.*
> 该市公布了一项宏大的计划，要在五年内建成新的公交系统。

和"一句话本质"的关系：这个计划要绕的圈太大了——周长惊人，走完需要很多脚步。

### 义项三：（带贬义的）野心勃勃的、权欲过重的

古老的阴影。在政治评论或某些带讽刺意味的语境中，ambitious 可以暗示"为了往上爬不择手段"。

> *He was too ambitious for his own good — he made enemies at every turn.*
> 他野心太重，反而害了自己——每走一步都在树敌。

和"一句话本质"的关系：绕得太狠了，把路上的人全踩了。

## 5. 裂缝标注

### 常见误用场景

**❌ 用 ambitious 形容"勤奋"**
ambitious ≠ hardworking。一个人可以很勤奋但没有野心（只想把眼前的活干好），也可以很 ambitious 但并不每天加班（他在策略性地布局）。Ambitious 的核心是**方向感和欲望**，不是工作量。

**❌ "an ambitious student" = 好学生？**
不完全是。Ambitious student 不是"认真学习的学生"，而是"有明确目标且想做大事的学生"。一个埋头刷题但没想过未来方向的学生是 diligent，不是 ambitious。

### 易混近义词辨析

| 词 | 核心区别 |
|---|---|
| **aspiring** | 强调"渴望成为某个身份"（aspiring writer = 想成为作家的人）。ambitious 更广，不限于某个具体身份 |
| **driven** | 强调内在驱动力的**强度**，不一定有具体目标。ambitious 总有一个明确的"那把椅子"在前方 |
| **aggressive** | 在商业语境中有时和 ambitious 互换（aggressive growth plan），但 aggressive 暗示的是**手段的强硬**，ambitious 暗示的是**目标的宏大** |
| **determined** | 侧重"不放弃"的韧性。ambitious 侧重"想要的东西很大"。你可以 determined to finish a puzzle（坚持拼完拼图），但这不叫 ambitious |

### 中文母语者陷阱

**最大的坑：ambitious = 野心 ≈ 贬义？**

中文里"有野心"经常带负面色彩（"这个人野心太大了"）。但英文里说 "She's ambitious" 在绝大多数语境下是**夸人**。如果你想表达贬义，需要额外加料：overly ambitious / ruthlessly ambitious / too ambitious for his own good。

**第二个坑：ambitious plan ≠ "有野心的计划"**

中文翻译成"宏大的""雄心勃勃的"都比"有野心的"准确。这个义项形容的是**事情的规模和难度**，不是拟人化的欲望。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境说明 |
|---|---|
| **ambitious plan / project / goal** | 最高频搭配。形容事情规模大、难度高。新闻报道和商业场景里满地都是 |
| **ambitious target** | 有挑战性的目标。常见于年报、KPI 讨论 |
| **overly ambitious** | "太贪心了"，暗示不切实际。带委婉批评意味 |
| **fiercely / hugely / wildly ambitious** | 程度副词加持，褒义为主 |
| **ambitious undertaking** | 偏正式/书面，形容一项宏大的事业。学术和新闻语体 |
| **politically ambitious** | 有政治抱负的。这个搭配里贬义概率略高——取决于说话人的态度 |
| **nothing if not ambitious** | 固定句式，"说别的不好说，但至少够大胆"。带一点欣赏一点调侃 |

### 语域标注

**全语域通用**。从日常对话（"You're so ambitious!"）到正式文件（"an ambitious legislative agenda"），这个词哪儿都能去。不存在"太正式"或"太口语"的问题。

但请注意：在**英式英语**的某些社交场合，直接说 "You're very ambitious" 可能被听成微妙的刺——英国人有时把 ambitious 当成"这人太想往上爬了"的委婉说法。美式英语里这个问题几乎不存在。

### 活用示范

> *The dessert menu was ambitious — seventeen items, each more architecturally improbable than the last.*
> 甜品单野心不小——十七道，一道比一道在建筑学上不可能。

——用 ambitious 形容一份菜单，幽默感来自把"雄心壮志"嫁接到甜品上。这种活用在非正式写作中很常见，体现的是 ambitious 的"规模惊人/胆子够大"义项。

---

## 7. 跨语言映射

中文在这个语义区间有一组词，微妙地切割了 ambitious 的领地：

- **"有野心"**：覆盖 ambitious 的贬义区间（太想要了、不安分），但在中文里比 ambitious 在英文里更偏负面
- **"有雄心 / 有抱负"**：覆盖 ambitious 的褒义区间，但听起来比 ambitious 更"高大上"、更书面。日常口语里你不太会说"这人很有抱负"
- **"上进"**：最接近日常口语中 ambitious 的褒义用法。"她很上进"≈ "She's quite ambitious"。但"上进"没有 ambitious 那种"盘子铺得很大"的规模感

日语里的**「野心的」（やしんてき）**和中文"有野心"一样偏贬义。褒义时日语更常用**「意欲的」**或**「志が高い」**。

有趣的是，拉丁语系语言（法语 *ambitieux*、西班牙语 *ambicioso*、意大利语 *ambizioso*）都保留了褒贬两面——这些词在各自语言里的感情色彩比英语的 ambitious 更摇摆，更接近它的罗马祖先。

---

**记忆钥匙**：下次看到 ambitious，别想"有野心的"——想那个穿白袍的罗马人，凌晨四点磨着鞋底在城里绕圈。那种停不下来的绕行，就是这个词两千年没变的骨骼。
