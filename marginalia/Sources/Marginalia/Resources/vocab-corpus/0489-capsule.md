# capsule

---

## 1. 一句话本质

**capsule 是"刚好够装下一个完整世界的最小容器"——不是大箱子，是把整颗药、整段历史、整个人压进去还能完好取出的那层壳。**

---

## 2. 词源考古

### 拆骨骼

```
capsule = caps- + -ule
         ↑        ↑
      capsa       拉丁语小词后缀（diminutive）
      "箱子"      → "小箱子"
         ↑
      capere
      "抓取、容纳"
         ↑
      PIE *kap-
      "抓"
```

**capsa**（箱子）→ **capsula**（小箱子）→ 法语 **capsule** → 英语 **capsule**

这个 `-ule` 后缀是拉丁语里特别温柔的一刀——把东西缩小。`glob-` → `globule`（小球），`mol-` → `molecule`（小质量）。所以 capsule 骨子里就是"小箱子"，但不是随便一个小盒子，而是"刚好装得下"的那种紧凑。

### 族谱

capsule 和你前几轮见过的词全是一家人——**PIE \*kap-（抓取）** 这条血脉：

| 词 | 和 *kap- 的关系 |
|---|---|
| **cap** | 抓住头顶的东西 |
| **capable** | 能抓住事情的 |
| **capacity** | 能抓住多少 |
| **capsize** | 存疑，可能是"头被抓翻了" |
| **capsule** | 抓住东西的小壳 |
| **capture** | 抓住了 |
| **captive** | 被抓住的人 |

### 历史流变

**17世纪**：英语从法语借入 *capsule*，最初就是字面意思——"小容器、小匣子"。

**18世纪**：植物学先下手了。植物学家用 capsule 命名种子荚——那种干了会裂开、把种子弹射出去的果实结构。这个用法精准到骨头里：种子荚就是植物造的"最小完整容器"，壳的存在只为保护内容物，使命完成就自毁。

**19世纪中期**：药剂师拿走了这个词。把苦药粉灌进明胶壳里让人能吞下去——**胶囊**诞生。这是 capsule 含义的关键转折点：**容器不只是保护内容物，还让内容物变得可接受**。苦的东西包上一层壳就能咽了。

**1950-60年代**：太空竞赛给了 capsule 最戏剧性的新生。把人类装进一个刚好够坐的金属壳里射进太空——**space capsule**。NASA 用这个词不是随便挑的：它精确传达了"把一切生存必需品压缩到最小体积"的工程哲学。

**1970年代至今**：capsule 开始抽象化。**capsule review**（极简书评）、**capsule collection**（胶囊系列/精简衣橱）、**time capsule**（时间胶囊）——核心语义完成了从物理到概念的跳跃：**把完整的东西压缩到最小形态**。

## 3. 故事 / 画面

1977 年，NASA 往旅行者号探测器上放了一张金色唱片。

一张唱片。115 张图片、55 种语言的问候、90 分钟音乐、一个小时的脑电波记录。整个人类文明——海浪声、巴赫、鲸鱼歌、一个母亲亲吻婴儿的声音——被压进一张 30 厘米的镀金铜盘里。

外面的铝壳刻着播放说明，用的是氢原子自旋频率当翻译密钥。设计寿命十亿年。

这就是 capsule 的终极形态：**一个壳，装着一个完整但被压缩的世界，穿越某种不可能的距离——时间的、空间的、或者苦味的**。

药用胶囊穿越的是舌头对苦味的抗拒。太空舱穿越的是大气层。时间胶囊穿越的是几十年的遗忘。旅行者号的金唱片穿越的是星际空间。

容器越小，里面的东西越完整，这个词就越准确。

## 4. 多面体：核心义项展开

### 义项一：物理小容器（药用胶囊）

> *Take two capsules with water before meals.*
> 饭前用水吞服两粒胶囊。

最日常的用法。注意 capsule 不等于 tablet（药片）——capsule 特指有壳的，里面是粉或液体；tablet 是压实的固体。壳的存在是 capsule 的灵魂。

**和本质的关系**：最字面的"最小容器装完整内容"——那层明胶壳刚好包住药粉，多一毫米都没有。

### 义项二：太空舱

> *The capsule splashed down in the Pacific Ocean after a six-month mission.*
> 执行六个月任务后，太空舱溅落在太平洋。

这里 capsule 的选择特别有味道——不叫 ship、不叫 vessel、不叫 craft。叫 capsule 是因为宇航员在里面几乎不能伸直腿。它强调的是**空间的极端压缩**。

**和本质的关系**：把维持人类生存的全部系统塞进一个勉强能坐的壳。

### 义项三：植物蒴果/种子荚

> *When the capsule dries, it splits open and scatters seeds across the ground.*
> 蒴果干燥后会裂开，把种子撒满地面。

植物学术语，但它揭示了 capsule 一个隐藏属性：**壳是临时的，是为了被打开或被消解**。药用胶囊在胃里溶解，种子荚裂开弹射，太空舱打开舱门——capsule 从来不是永久封闭的。

**和本质的关系**：容器的使命是保护内容物到达目的地，然后释放。

### 义项四：形容词化——"浓缩的、精要的"

> *Here's a capsule history of the French Revolution in under 500 words.*
> 这是一份 500 字以内的法国大革命极简史。

这是 capsule 最抽象的用法，也是最优雅的语义延伸。capsule review、capsule summary、capsule biography——都指"把完整内容压缩到极小篇幅但不丢失本质"。

**和本质的关系**：从物理压缩到信息压缩，"最小容器装完整内容"的逻辑完全一致。

### 义项五：时间胶囊

> *They buried a time capsule beneath the school, to be opened in fifty years.*
> 他们在学校地下埋了一个时间胶囊，五十年后开启。

time capsule 之所以用 capsule 而不是 box 或 container，是因为它要穿越时间——这层壳保护的不是空间距离，是时间距离。

**和本质的关系**：把"现在"的一个截面完整封存，让它穿越到未来。

## 5. 裂缝标注

### 常见误用

**❌ 把 capsule 和 tablet 混用**

药学语境下这是硬错。capsule 有壳（gelatin shell），里面是粉末或液体；tablet 是药粉压成的固体块。日常口语混用不会被纠正，但写医学/健康类文本会显得不专业。

**❌ capsule collection ≠ "一个小系列"**

时尚界的 capsule collection 不只是"小"——它的核心是**件件可互搭、去掉任何一件就不完整**。一个品牌出 20 件不相干的衣服不叫 capsule collection，那叫 small collection。capsule 的精髓是压缩后的完整性。

### 易混近义词

| 词 | 区别 |
|---|---|
| **container** | 通用容器，任何大小形状。capsule 特指小而紧凑、刚好包住内容物的 |
| **pod** | 更强调密封独立空间（escape pod, sleeping pod）。capsule 更强调"被压缩" |
| **shell** | 只指壳本身。capsule 包含壳+内容物的整体 |
| **tablet** | 药片=压实固体，无壳。capsule=有壳有内容 |

### 中文母语者陷阱

**"胶囊"的中文语义比英文窄。** 中文里"胶囊"几乎只让人想到药用胶囊和胶囊旅馆。但英文 capsule 的覆盖面大得多——太空舱、浓缩书评、精简衣橱、时间胶囊——这些用法中文母语者往往想不到用"胶囊"去对应。

**"太空舱"的"舱"会误导。** 中文的"舱"暗示一个相当大的空间（船舱、机舱）。但 capsule 恰恰强调空间小到极限。脑子里别想"舱"，想"壳"。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境 |
|---|---|
| **time capsule** | 埋在地下、几十年后开的封存容器。也被苹果用作产品名（Time Capsule 路由器/备份盘） |
| **space capsule** | 载人航天器，特指返回舱这种紧凑型 |
| **capsule collection** | 时尚界：精简互搭的小系列。也引申到 capsule wardrobe（极简衣橱） |
| **capsule review/summary** | 新闻/出版：极短的浓缩评论或摘要 |
| **encapsulate** | 动词化，"封装、概括"——编程里的 encapsulation（封装）就是这个词 |
| **capsule hotel** | 胶囊旅馆，日本发明的极小住宿空间 |
| **capsule endoscopy** | 胶囊内窥镜——吞一颗带摄像头的胶囊 |

### 语域

capsule 是个跨语域词——日常（吃胶囊）、学术（植物学蒴果）、新闻（capsule review）、科技（space capsule）、时尚（capsule collection）、编程（encapsulation）全能用。没有正式程度限制，不算俚语也不算过于书面。

### 活用示范

> *She gave a capsule version of her life story—born in Shanghai, raised in Vancouver, landed in Berlin, never looked back.*
> 她给了一个胶囊版人生简史——上海出生，温哥华长大，落脚柏林，从未回头。

这里 capsule 当形容词用（capsule version），传达的是"完整人生被压到一句话，但什么都没丢"。这种用法在口语和非虚构写作中很有表现力。

> *The whole argument was encapsulated in a single chart.*
> 整个论点被封装在一张图里。

encapsulate 是 capsule 最活跃的动词形态。编程之外的日常用法也极其常见。

## 7. 跨语言映射

**日语「カプセル」**（kapuseru）直接音译，用法几乎和英语一一对应：カプセルホテル（胶囊旅馆）、タイムカプセル（时间胶囊）。但日语额外贡献了一个文化意象——日本的胶囊旅馆把 capsule 的"最小完整空间"推到了人类居住的极限。

**编程中的 encapsulation**：面向对象编程里的"封装"概念和 capsule 的词源高度同构——把数据和方法装进一个类的"壳"里，外面只能通过接口访问。壳保护内容物，内容物在壳里是完整的。如果你学过 OOP，这可能是最好的记忆锚点。

**中文"胶囊"**：有趣的是，中文选了"胶"（材质）+"囊"（袋子）来翻译，锚定的是药用胶囊的物理形态。而英文 capsule 的词源锚定的是"小容器"的抽象概念——所以英文能自然延伸到太空舱、浓缩评论，中文的"胶囊"在这些语境下就不太自然（我们说"太空舱"不说"太空胶囊"）。翻译时的选择锁死了后续的语义扩展空间——这本身就是一个语言学上有意思的现象。
