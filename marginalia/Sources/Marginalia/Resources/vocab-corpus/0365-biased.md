# biased

---

## 1. 一句话本质

**biased 是一颗被灌了铅的保龄球——它看起来圆，但永远不会走直线，因为重心早就偏了。**

你以为它在公平地滚，其实出手的那一刻，方向已经定了。

---

## 2. 词源考古

### 词根词缀拆解

**bias**（名词/形容词）+ **-ed**（过去分词后缀，表"被施加了某种状态"）

核心在 **bias** 本身。

- 来自 **古法语 biais**，意思是 **"斜的、倾斜的、不正的"**（oblique, slanting）
- 更早的来源 **存疑**，学界两条主要假说：
  - **假说一**（较主流）：来自古普罗旺斯语 biais，可能追溯到通俗拉丁语 ***biaxius**——拆开就是 **bi-**（两个）+ **axis**（轴）。字面意思："有两根轴的"。两根轴，重心就不在正中，所以歪了。
  - **假说二**：与希腊语 **epikarsios**（ἐπικάρσιος，斜向的）有关联，但语音演变链条不完整，争议较大。

**⚠️ 注意**：那个流传甚广的"bi- + axis → 两条轴所以偏了"的词源故事很漂亮，但语言学界并未完全定论。标注为"可能但未确证"。

### 历史流变：从草坪保龄球到认知偏差

这个词的历史有一个极其精彩的具象→抽象转变：

**16世纪（草坪保龄球时代）**

bias 最初进入英语，是一个**保龄球术语**。当时英国流行 lawn bowls（草坪滚球），球不是正圆的——工匠会故意在球的一侧**灌入更重的材料**，让球的重心偏移。这个被灌了铅的一侧就叫 **the bias**。于是球滚出去的时候会走一条弧线，而不是直线。

这不是缺陷，是**设计**。玩家要利用这个偏移来绕过障碍球。

**17世纪（转喻：从球到人）**

人们开始用 bias 形容人的思维——"他的判断有 bias"，就像一颗被灌了铅的球，看着在滚，其实方向早就被内部的重量决定了。

这个比喻太精准了，以至于原来的保龄球含义几乎被遗忘，抽象义彻底上位。

**20世纪（统计学、电子学、AI）**

- **统计学**：sampling bias（样本偏差）——你的数据不是随机的，某一类被多选了
- **电子学**：bias voltage（偏置电压）——给电路施加一个预设的"偏移量"
- **21世纪 AI**：algorithmic bias——训练数据里的偏见被模型继承了

**文化地层总结**：这个词从一个物理上可以触摸到的"铅块"，变成了一个你看不见但无处不在的"认知倾斜"。而它的内核从未变过——**重心不在中间。**

## 3. 故事 / 画面

一张法官的桌子。

桌面看起来很平。但如果你蹲下来从侧面看，会发现桌脚有一只被垫高了 3 毫米。桌上的弹珠会慢慢地、不知不觉地向一个角落滚去。

坐在桌子后面的法官全程没有推弹珠。他甚至不知道桌脚被垫过。弹珠自己滚过去的。他真诚地认为自己很公平。

**这就是 biased 最阴险的一面：它不需要你故意作恶，只需要你脚下的地面本身就不平。**

在你察觉之前，所有的球已经滚向了同一个角落。

## 4. 多面体：核心义项展开

### 义项一：有偏见的、带有倾向性的（最常用）

> *The article was clearly biased in favor of the ruling party.*
> 这篇文章明显偏向执政党。

和"一句话本质"的关系：你的判断被内部的"铅块"（先入之见、立场、利益）带偏了弧线。

### 义项二：统计上有偏的、样本不具代表性的

> *The survey results are biased because they only sampled urban residents.*
> 调查结果有偏差，因为只抽样了城市居民。

和"一句话本质"的关系：数据这颗"球"的构成不均匀——某些群体被多装了，某些被少装了，所以滚出来的结论天然歪。

### 义项三：（电子学）施加了偏置的

> *The transistor must be properly biased to operate in the active region.*
> 晶体管必须正确偏置才能工作在有源区。

和"一句话本质"的关系：人为给系统一个预设的偏移量，让它在你想要的区间工作。这里 biased 是**中性甚至正面**的——偏置是**设计要求**。这反倒接近了 bias 最原初的保龄球含义：偏移不是错，是工具。

### 义项四：（对某事物）偏爱的、有好感的（biased toward/towards）

> *I'm biased toward hiring candidates with startup experience.*
> 我偏向于录用有创业经验的候选人。

和"一句话本质"的关系：不是说你不公平，是承认你心里那颗球的重心确实偏了一边。这个义项常用于**自我坦白**，语气比 prejudiced 轻很多。

## 5. 裂缝标注

### 常见误用场景

**❌ 把 biased 当作"客观上错误"来用**

biased 不等于 wrong。一个研究可以有 bias 但结论仍然是对的；一个人可以 biased toward 某种方案但那个方案确实最优。bias 描述的是**过程的倾斜**，不是**结果的错误**。

**❌ biased 后面的介词乱用**

- biased **toward / towards** = 偏向、偏爱
- biased **against** = 偏见、歧视
- biased **in favor of** = 偏向（更正式）

不能说 ~~biased to~~（做"偏向某事"用时），这是中文"对…有偏见"的直译陷阱。

### 易混近义词辨析

| 词 | 微妙差别 |
|---|---|
| **biased** | 重心偏了，可能自己不知道。可以是轻微的、无意识的 |
| **prejudiced** | 更重——"预先就判了"，通常指基于种族/性别/阶级的偏见，道德谴责感更强 |
| **partial** | 在"偏心"的意义上比 biased 文雅，常见于法律/正式文体。*a partial judge* = 一个偏心的法官 |
| **one-sided** | 强调只展示了一面。一个 biased 的报道是内部倾斜的，一个 one-sided 的报道是**结构上缺了一半** |
| **skewed** | 更技术化，常用于数据/统计。强调分布形状的不对称 |

**场景对比**：

- 新闻编辑选择性报道 → **biased** reporting（最自然）
- 法庭上一方觉得法官偏心 → **partial** 或 **biased**
- 朋友推荐餐厅，自曝"我是那家的常客" → "I'm **biased**"（轻松自嘲）
- 同样的场景你说 "I'm **prejudiced** toward that restaurant" → 极其怪异，prejudiced 太重了

### 中文母语者陷阱

**"偏见"和 biased 不完全对应。**

中文"有偏见"几乎总是贬义的、道德化的。但英文 biased 可以很中性甚至带自嘲的幽默：

> "I'm a little biased — I grew up in that city."

这句话翻译成"我有点偏见"在中文里太严肃了。更准确的中文语感是"我不够客观啦，毕竟我就是那儿长大的"。

**另一个陷阱：** 中文"偏向"可以是中性的策略选择（"我偏向方案 A"），但英文说 "I'm biased toward Plan A" 会被听成"我不够客观"，而不只是"我倾向于"。想表达纯粹的策略倾向，用 **lean toward** 或 **prefer** 更安全。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境说明 |
|---|---|
| **biased against** minorities/women/... | 社会议题中最常见的结构，指向歧视 |
| **biased toward(s)** a particular outcome | 实验/判断偏向某个结果，学术和日常都用 |
| **politically biased** | 新闻/媒体批评中的高频词组 |
| **culturally biased** | 测试/评估工具对某些文化群体不公平 |
| **inherently biased** | 强调偏见是内建的、结构性的，不是个人选择 |
| **biased sample / biased data** | 统计学标配 |
| **biased reporting / biased coverage** | 新闻学标配 |

### 语域标注

**全语域通行证。** biased 是少数能在几乎所有场合使用的词：

- 学术论文 ✅（*The study's methodology was biased toward...*）
- 日常吐槽 ✅（*The referee was so biased!*）
- 法律语境 ✅（*The jury was biased.*）
- 自嘲 ✅（*OK, I might be biased, but...*）
- AI/科技 ✅（*The model is biased against...*）

正式程度：中等偏上，但不排斥口语。比 prejudiced 温和，比 unfair 精准。

### 活用示范

> *"I'm biased, but I think this is the best pizza in the city." — Every local ever*

这种用法是一种修辞免责声明。先承认"我不客观"，然后给出一个充满感情的判断。效果是让听者觉得你很坦诚，反而更愿意相信你。

> *"History is written by the victors — and it's always biased."*

这里 biased 不是在骂谁，而是在陈述一个结构性事实：叙事者的位置决定了叙事的弧线。

## 7. 跨语言映射

**中文"偏见"** vs. **biased**

中文"偏见"的"偏"字本身就是空间隐喻——偏离中心。和 bias 的保龄球词源异曲同工。但中文"偏见"几乎锁死在"不好的、应该纠正的认知错误"这个语义区间里，而英文 biased 的语义弹性大得多——可以是自嘲、可以是技术描述、可以是中性陈述。

**日文「偏る」（かたよる）**

字面是"偏向一方"，比中文"偏见"更接近 biased 的中性感。日文可以说"味が偏っている"（口味偏了），不带道德判断，纯粹描述倾斜。这个质感更接近英文 biased 的温度。

**法文 biaisé**

直接从同一个词源来的，但在法文里日常使用频率远低于英文，更偏书面和技术用途。法文日常说"有偏见"更常用 **partial** 或 **de parti pris**。

---

**最后一颗钉子：** 下次你听到有人说 "I might be biased, but..."，脑子里浮现那颗被灌了铅的保龄球就对了——它马上要告诉你一条弧线，而这条弧线的方向，在它出手之前就已经写好了。
