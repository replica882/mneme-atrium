# adequate

---

## 1. 一句话本质

adequate 是一条刚好齐平的水位线——水没漫过来，但你也看不见多余的一滴。它是"够了"，但这个"够"字里藏着一声叹息。

---

## 2. 词源考古

**拆骨骼：**

```
ad- （朝向，到达） + aequ- （平、等） + -ate （动词/形容词后缀）
      ↓                    ↓
   拉丁语 ad            拉丁语 aequus（平的、相等的）
```

原始拉丁语动词是 **adaequāre**——"使之与……齐平"。过去分词 **adaequātus** 进入英语，变成了形容词 adequate。

所以这个词的骨骼在说一件很具体的事：**把一个东西拉到跟标准线一样高**。不是超过，是齐平。

**历史流变：**

- **1610s** 左右进入英语，直接从拉丁语 *adaequātus* 借来。最初是个中性甚至偏正面的词——"与任务相称的、胜任的"。
- **17–18 世纪**，在哲学和法律文本中大量使用，指"充分的、足够的"，没有贬义。"adequate cause"（充分原因）、"adequate remedy"（充分救济）都是正经术语。
- **19 世纪末–20 世纪**，关键漂移发生了：随着英语中表达"好"的词越来越卷（excellent, outstanding, exceptional 轮番上阵），adequate 被挤到了评价光谱的下半段。**它没变，是周围的词把它显得矮了。**
- **当代英语**：说一个人 adequate，基本就是在说"嗯……及格吧"。这个词已经带上了一层不说破的失望感。

**文化地层：**

这是个典型的"通货膨胀受害者"。在一个 amazing、incredible、fantastic 都被用来形容一顿还行的午饭的语言环境里，adequate 守住了它的本义——"够了"。但正因为别人都在通胀，它的购买力缩水了。你说一部电影 adequate，等于在说"我不会要求退票，但我也不会推荐给朋友"。

同根词 **equal**（aequus → equal）、**equate**（使相等）、**equation**（等式）、**equity**（公平）——整个 aequ- 家族都在讲"平"这件事。adequate 是这个家族里最不起眼的那个成员：它永远刚好齐平，从不冒尖。

## 3. 故事 / 画面

学校体测，跳高。

横杆设在 1.2 米。你助跑、起跳，身体刚好擦过横杆，杆晃了一下——没掉。

你落在垫子上。考官在表格上打了个勾。没掌声，没失望，没任何多余的表情。

这就是 adequate。

横杆没掉 = 标准达到了。但你的后背几乎是贴着杆过去的，一根头发的余量都没给。考官的那个勾，打得毫无感情——因为 adequate 本来就不是用来激动的词。

## 4. 多面体：核心义项展开

### 义项 A：足够的、充分的（量/质达标）

> *The funding was adequate for the first phase, but we'll need more later.*
> 资金够应付第一阶段，但后面得追加。

和"一句话本质"的关系：水位线刚好齐平——够用，但没有溢出。

### 义项 B：胜任的、合格的（人/能力达标）

> *She's an adequate programmer, but she's not the one you'd call at 3 a.m. when the server goes down.*
> 她写代码是够格的，但凌晨三点服务器炸了你不会打给她。

这个义项最能暴露 adequate 的暗面。用在人身上时，"齐平"就变成了"也就那样"。你不会在推荐信里写 adequate——除非你想暗杀这个人的前途。

### 义项 C：（法律/技术）充分的、适当的

> *The court found that adequate notice had been given to all parties.*
> 法院认定已向各方发出了充分通知。

在法律和技术语境里，adequate 回到了它最古老的中性含义：达到了规定标准。这里没有失望感，纯粹是技术判定。

**底层统一性：** 三个义项共享同一个骨骼——**与某条线齐平**。区别只在于那条线是钱的标准、人的标准，还是法律的标准。

## 5. 裂缝标注

### 常见误用场景

**❌ 用 adequate 当夸奖。**

> "Your presentation was adequate." ← 你以为在说"很好"，对方听到的是"及格而已"。

在当代日常英语里，adequate 带有 damning with faint praise（以微弱的赞美来贬低）的效果。如果你真觉得不错，用 solid、good、impressive。adequate 留给你想表达"过得去但不突出"的场合。

### 易混近义词辨析

| 词 | 核心区别 |
|---|---|
| **sufficient** | 纯客观量够了。"sufficient evidence"（证据够了）。没有 adequate 那层"也就这样"的情绪色彩。更冷静、更技术。 |
| **enough** | 日常口语版的 sufficient。"Is this enough?"——随口问一嘴，没人会觉得你在暗讽。 |
| **satisfactory** | 也是"达标"，但比 adequate 暖半度。satisfactory 暗示"检查了，通过了，行"；adequate 暗示"凑合着也行吧"。 |
| **competent** | 专指能力。"a competent doctor" 是"靠谱的医生"；"an adequate doctor" 是"不至于医死你的医生"——你感受一下这个温差。 |

### 中文母语者陷阱

**"adequate = 足够的"这个等号画得太快了。**

中文的"足够"是中性偏正面的词——"你做得足够好了"是安慰，不是贬低。但 adequate 在很多语境里自带一层凉意。如果你想表达中文里那个温暖的"足够了"，用 more than enough / plenty 更安全。

另一个坑：中文说"你的能力是足够的"可能是在肯定对方，但 "Your ability is adequate" 在英语里听起来像在暗示"但也仅此而已"。

## 6. 搭配与语域

### 高频搭配

| 搭配 | 语境 |
|---|---|
| **adequate supply / resources** | 物资/资源够用——新闻、报告常见 |
| **adequate protection** | 保护措施到位——法律、安全领域 |
| **adequate preparation** | 准备充分——考试、项目管理 |
| **adequate time** | 时间够——正式通知、合同 |
| **adequate compensation** | 合理补偿——法律、HR |
| **prove adequate** | 证明够用——叙事性文本，常出现在转折前："initially proved adequate, but..." |
| **barely adequate** | 勉强够——加了 barely 后那层凉意直接变成了冷风 |

### 语域标注

偏**书面和正式**。学术论文、法律文书、新闻报道、商业邮件中高频出现。日常口语中不太会说 "this is adequate"——更可能说 "this is fine" 或 "this'll do"。

如果在口语中刻意用 adequate，往往是在**反讽**——"Oh, the service was... adequate" 配上一个表情，你懂的。

### 活用示范

> *"How was the hotel?" "Let's just say it was adequate and leave it at that."*
> "酒店怎么样？""就说它 adequate 吧，别的不提了。"

这句话的杀伤力在于 "leave it at that"——adequate 加上一句"说到这儿吧"，听的人立刻知道：差，但不值得浪费口水去骂。

> *The dam held. Barely adequate by modern standards, but it held.*
> 大坝挺住了。按现代标准看勉强合格，但它挺住了。

这句里 adequate 和叙事张力配合得很好——"勉强够"本身构成了戏剧性。

## 7. 跨语言映射

中文里最接近 adequate 这种"够了但你也别期待更多"温度的词，可能是**"凑合"**。

"凑合"比 adequate 更口语、更随意，但那层"不好不坏、将就着用"的情绪是对的。"这饭凑合能吃"≈ "The food is adequate"——都是齐平线上的评价，都带着一声不太明显的叹息。

日语的「まあまあ」（maa maa）也踩在类似的位置——"嗯，也行吧"。但「まあまあ」的语气比 adequate 更柔，更像是在安慰自己，而 adequate 更像是在写报告。

拉丁语系的近亲们——法语 *adéquat*、西班牙语 *adecuado*——都比英语的 adequate 活得更体面。法语里说 *une réponse adéquate*（一个恰当的回答）是正面评价，没有英语里那层"也就这样"的暗伤。这再次说明：**不是 adequate 自己变了，是英语的评价通胀把它踩低了。**
